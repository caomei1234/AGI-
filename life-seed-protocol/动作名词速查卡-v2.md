# 动作名词速查卡（贝贝版）

> 贝贝做题卡住了 → 看这张卡 → 找到那个词 → 换一个说法 → 秒懂。
> 每个词都有一个贝贝最熟悉的场景。先做动作，再做题。

---

## 🦴 骨头1：比一下

| 教科书说的 | 动作名词 | 怎么跟贝贝说 | 贝贝最熟悉的场景 |
|-----------|---------|-------------|----------------|
| 大于/小于 | 多出来/少下去 | "你碗里多出来几个，我碗里少下去几个" | 两碗糖果放桌上，看哪碗多 |
| 绝对值 | 离零多远 | "不看你往左走还是往右走，只看你走了多远" | 站在家门口，不管往左走还是往右走，走了10步就是离家10步 |
| 因数 | 能整除的数 | "12能被3整除，3就是12的因数——3是12的一个零件" | 12块积木，能刚好分成3堆吗？能→3就是因数 |
| 倍数 | 乘出来的数 | "3×4=12，12就是3的倍数——12是3长出来的" | 每堆3块积木，堆了4堆，一共12块——12是3长出来的 |
| 质数 | 只有一种拆法的数 | "7只能拆成1×7——它很倔" | 7块积木，只能分成1堆7块或7堆1块，没有别的分法 |
| 合数 | 有好几种拆法的数 | "12能拆成2×6、3×4——它很灵活" | 12块积木，能分成2堆6块、3堆4块、4堆3块——好多分法 |
| 最大公因数 | 两个数共同的最大的尺子 | "12和18共同的最大的尺子是6" | 12块积木和18块积木，都能刚好用同一种堆法分完，最大的那种堆是几块一堆？ |
| 最小公倍数 | 两个数第一次碰面的地方 | "4和6最小公倍数是12" | 你每数4个拍一下手，我每数6个拍一下手，数到第几个我们同时拍？第12个 |
| 奇数/偶数 | 分不成两半/能分成两半 | "7个糖分给2个人分不均匀——奇数" | 7颗糖分给你和我，能一人一样多吗？不能→奇数。8颗呢？能→偶数 |
| 排序 | 从大到小排 | "最高的站第一个，最矮的站最后" | 小朋友排队比身高，从高到矮站好 |

| 比例尺 | 图上一小段等于实际一大段 | "图上1厘米等于实际1公里" | 地图上你指甲盖那么长，实际要走好远好远 |
| 概率 | 全部里摸到想要的有多可能 | "袋子里3红7白，摸到红的——十份里取三份" | 口袋里3颗红色糖7颗白色糖🍬，闭眼摸一颗，摸到红的有多可能？ |
| 密度 | 一份有多重 | "一立方厘米的铁有多重" | 同样大小的两块东西，铁的比木头的重——铁密度大 |
| 速度 | 一份时间走多远 | "一秒钟走多少米" | 你跑步🏃一秒钟跑了多远？ |
| 浓度 | 一份里有多少是糖 | "糖水里十份有几份是糖" | 一杯糖水，尝一口——甜不甜？甜=糖多=浓度高 |
| 折扣 | 原价的几份儿取几 | "打八折——原价的十份儿取八份" | 玩具原来100块，现在只要80块——打了八折 |

| 利率 | 本金的几份儿取几是利息 | "100块存一年生5块利息" | 你把100块存银行🏦，一年后多了5块——那5块就是利息 |
| 平均数 | 总数÷份数 | "把所有数加起来除以有几个数" | 你考了80分我考了100分，平均分=(80+100)÷2=90分 |
| 短几分之几 | 切出差距那段看占整体多少 | "你比我矮的那段，占你身高的几份儿取几" | 你身高1米2，我身高1米，你比我高20厘米——那20厘米占你1米2的几份儿取几？ |

## 🦴 骨头3：倒着推

| 教科书说的 | 动作名词 | 怎么跟贝贝说 | 贝贝最熟悉的场景 |
|-----------|---------|-------------|----------------|
| 方程 | 用等式逼出未知数 | "□+3=8，□是多少？逼出来" | 跷跷板——左边放了□+3，右边放了8，两边一样重，□是几？ |
| 一元一次方程 | 一个未知数，逼一次 | "只有一□，逼一次就出来了" | 一个□，一个条件，逼一次就知道了 |
| 因式分解 | 展开的倒回去 | "积木拼好了=展开，积木拆开=因式分解" | 乐高拼好了一辆车🚗——展开。把车拆回一块块积木——因式分解 |
| 不等式 | 不是等于，是大于或小于 | "不是刚好，是多了或者少了" | 跷跷板——不是平的，一边高一边低 |
| 移项 | 搬家 | "+3从左边搬到右边变成-3" | 你碗里有3颗糖要移到我碗里——你少了3我多了3 |
| 充分条件 | 有它就行 | "有钥匙就能开门🔑" | 有钥匙🔑就能开门——钥匙是充分条件 |
| 必要条件 | 没它不行 | "没电手机就不能用🔋" | 手机没电了🔋就不能用——电是必要条件 |

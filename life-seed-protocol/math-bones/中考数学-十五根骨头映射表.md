# 中考数学知识点 → 十五根骨头 · 映射表

作者：丁玲 & 岚
2026-06-29

> 拿到任何一道中考题，先问：底下是哪根骨头？
> 认出骨头——不用背公式。认不出——衣服太厚，往回剥。

---

## 使用说明

这张表做两件事：

1. **正向查**：女儿做错一道题 → 查这道题考什么知识点 → 查这个知识点底下是哪根骨头 → 找到断裂点 → 修
2. **反向练**：从第一根骨头开始 → 把这根骨头对应的所有真题拎出来 → 一口气做完 → 这根骨头扎稳了 → 换下一根

**骨头顺序不能跳。** 前面的骨头是后面的地基。断了前面的，后面的就悬空。

## 第一组：比较（地基·最重要）

> 一切数学从"能分辨不同"开始。
> 比较是数学的C大调。do。一比较万物生。

### 骨头1：直接比较——你几个我几个

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 有理数（整数、分数、小数） | 直接比较"有多少" | 比大小、数轴上标位置 |
| 数轴 | 比较的可视化——位置就是比较 | 数轴上的点表示数、比较大小 |
| 相反数 | 跟0的距离一样、方向相反 | 求相反数、化简 |
| 绝对值 | 比较的时候不看方向，只看距离 | 去绝对值符号、化简 |
| 倒数 | 乘积为1的两个数——比率的逆 | 求倒数 |
| 科学记数法 | 大数比较的简便写法 | 用科学记数法表示 |
| 近似数/精确度 | 比较的精度——能精确到哪一位 | 四舍五入、有效数字 |
| 实数（无理数、有理数） | 所有能比较大小的数 | 实数分类、比大小 |
| 二次根式 | 根号下的数——比较的扩展 | 化简、比大小 |

**17件衣服。一根骨头。**

训练方法：给女儿一道"比大小"的题。不管穿什么衣服（有理数、无理数、根式、绝对值）——底下都是"直接比较"。她认出来了——通了。

---

### 骨头2：比率比较——每一份多大

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 比（3:5） | 两个东西直接比较 | 化简比、求比值 |
| 比例 | 比率的关系不变 | 解比例、判断是否成比例 |
| 百分数 | 比率，分母固定100 | 百分数与小数分数互化 |
| 折扣/税率/利率 | 百分数的衣服，商业场景 | 打折问题、利息计算 |
| 浓度/密度/速度 | 比率，不同单位 | 行程问题、浓度问题 |
| 频率 | 比率——发生了几次÷总共几次 | 用频率估计概率 |
| 相似比 | 形状组的比率——对应边之比 | 求相似比、面积比 |
| 锐角三角函数值 | sin/cos/tan——旋转后的比率 | 求三角函数值 |

**8件衣服。一根骨头。**

训练方法：比率这根骨头贯穿代数和几何。"折扣"和"相似比"看起来是两个东西——底下都是"每一份多大"。女儿认出"相似比也是比率"——几何和代数就连上了。

---

### 骨头3：逆向比较——知道答案倒回去找

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 一元一次方程 | 已知结果求原因——逆向比较 | 解方程、应用题 |
| 二元一次方程组 | 两个逆向比较同时做 | 解方程组、应用题 |
| 一元二次方程 | 逆向比较，但答案可能有两个 | 解方程（公式法、配方法、因式分解） |
| 分式方程 | 逆向比较，但除法里有未知数 | 解分式方程、验根 |
| 一元一次不等式 | 比较的范围，不是精确值 | 解不等式、在数轴上表示 |
| 一元一次不等式组 | 两个范围取交集 | 解不等式组 |
| 列方程解应用题 | 把文字翻译成逆向比较 | 审题→列方程→解→检验 |

**7件衣服。一根骨头。**

训练方法：方程的本质是"逆向比较"——已知结果，倒回去找原因。女儿卡在"列方程"上——不是不会解方程，是不知道怎么把文字翻译成逆向比较。问她："这道题的结果是什么？你知道什么条件？倒回去。"

## 第二组：变化

> 先能比多少，才能比变了没有。

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 乘方 | 重复的乘法 | 求幂、化简 |
| 整式乘法 | 重复的分配律 | 单项式×多项式、多项式×多项式 |
| 因式分解 | 乘法的逆——拆出重复的因子 | 提公因式、公式法 |
| 科学记数法（大数） | 重复的10连乘 | 表示大数 |
| 数列（找规律） | 重复的规律 | 找第n个数、求和 |

**5件衣服。一根骨头。**

### 骨头5：绑定——两个量绑在一起

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 正比例函数 y=kx | 两个量绑在一起，比不变 | 画图、求解析式 |
| 反比例函数 y=k/x | 两个量绑在一起，乘积不变 | 画图、求解析式 |
| 一次函数 y=kx+b | 直线绑定——每走一步高多少 | 画图、求解析式、交点 |
| 二次函数 y=ax²+bx+c | 抛物线绑定——加速变化 | 画图、顶点、对称轴、最值 |
| 锐角三角函数（变化视角） | 角度变→sin/cos/tan跟着变 | 已知角度求值、已知值求角度 |

**5件衣服。一根骨头。**

训练方法：一次函数和二次函数的区别——不是"公式不同"。是"绑定的方式不同"。一次函数：每一步变化量相同（匀速）。二次函数：每一步变化量在变（加速）。问女儿："你走路每步一样长——一次函数。你跑步越跑越快——二次函数。区别在哪？"

| 工程问题 | 快慢——每天做多少 | 列方程解应用题 |

**4件衣服。一根骨头。**

## 第三组：位置

> 先能比大小，再能比变化，才能标出点在哪。

| 点的对称（关于坐标轴/原点） | 名字的规律变化 | 求对称点坐标 |
| 中点坐标 | 两个名字取平均 | 求中点 |
| 函数图像上的点 | 点的坐标满足函数关系 | 判断点是否在图像上 |

### 骨头9：旋转——绕着转

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 旋转（图形变换） | 绕一个点转一个角度 | 画旋转后的图形、求坐标 |

## 第四组：形状

> 先知道点在哪，才能连成线、围成面。

### 骨头10：直的——两点之间最短

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 线段/射线/直线 | 最短的路的不同形态 | 概念辨析 |
| 平行线 | 两条最短的路永远不碰 | 判定、性质（同位角、内错角、同旁内角） |
| 垂线 | 两条最短的路交叉成直角 | 垂线段最短、点到直线距离 |
| 三角形（边、角） | 三条最短的路围起来 | 三角形性质、内角和 |
| 勾股定理 | 直角三角形三边的约束 | 求边长、判断直角三角形 |
| 等腰/等边三角形 | 三角形的特殊——两边相等/三边相等 | 求角度、证明 |
| 多边形内角和 | 多条最短的路围起来 | 求内角和、外角和 |
| 平行四边形 | 两组对边分别平行 | 性质、判定 |
| 矩形/菱形/正方形 | 平行四边形的特殊衣服 | 性质、判定、面积 |
| 梯形 | 只有一组对边平行 | 面积、中位线 |

**10件衣服。一根骨头。**

训练方法：所有四边形——平行四边形、矩形、菱形、正方形、梯形——都是"直的"这根骨头的衣服。区别在于"约束条件不同"。问女儿："平行四边形的约束是什么？两组对边平行。矩形呢？两组对边平行+一个直角。正方形呢？两组对边平行+四个直角+四边相等。约束越多，形状越特殊。"

---

### 骨头11：圆的——到一个点的距离都一样

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 圆的基本性质 | 到圆心距离都一样 | 半径、直径、弦 |
| 垂径定理 | 直径⊥弦→平分弦和弧 | 求弦长、弧长 |
| 圆心角/圆周角 | 转了多少度（旋转组在这里又出现了） | 求角度 |
| 弧长/扇形面积 | 圆的一部分——按角度切 | 求弧长、面积 |
| 圆与直线的位置关系 | 圆和直线的距离vs半径 | 相离、相切、相交 |
| 切线 | 刚好碰到圆——距离=半径 | 判定、性质 |
| 圆与圆的位置关系 | 两个圆的距离vs两个半径 | 外离、外切、相交、内切、内含 |
| 正多边形与圆 | 圆内接/外切多边形 | 求边长、面积 |

**8件衣服。一根骨头。**

---

### 骨头12：对称——动了还跟没动一样

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 轴对称 | 翻过去还一样 | 画对称图形、求对称点 |
| 中心对称 | 转180度还一样 | 判断中心对称图形 |
| 全等三角形 | 移动后完全重合 | 判定（SSS、SAS、ASA、AAS、HL）、证明 |
| 相似三角形 | 等比放大后形状一样 | 判定、求比例、面积比 |
| 位似 | 相似的特殊情况——从一个点放大 | 求位似中心、画图 |
| 图形的旋转对称 | 转某个角度还一样 | 旋转对称图形辨认 |

**6件衣服。一根骨头。**

训练方法：全等和相似——底下都是"对称"。全等=动了完全重合（大小不变）。相似=动了形状一样（大小变了）。问女儿："全等是复印。相似是放大缩小后的复印。区别在哪？大小。但形状没变。形状没变=对称。"

## 第五组：推理

> 前四组是造房子。推理是检查房子有没有裂缝。

### 骨头13：无限接近——差多少我说了算

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 极限（初步） | 差多少我说了算 | 数列的趋势 |
| 二次函数的最值 | 无限接近最高/最低点 | 求最大值、最小值 |
| 切线（极限视角） | 割线两端无限靠近→切线 | 理解切线的本质 |

**3件衣服。一根骨头。**

中考对极限考得不多，但二次函数的最值本质上就是"无限接近"——抛物线的顶点是所有点里最高或最低的那个。

---

### 骨头14：从已知推未知——用你知道的推出你不知道的

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 命题/逆命题/否命题/逆否命题 | 推理的方向——正着推和反着推 | 判断四种命题的真假 |
| 证明（几何） | 从已知条件一步步推出结论 | 写证明过程 |
| 全等证明 | 从已知边角推出全等 | SSS、SAS、ASA、AAS、HL |
| 相似证明 | 从已知条件推出相似 | 判定、写比例 |
| 辅助线 | 给推理搭桥——补一条线让已知和未知连上 | 画辅助线、写理由 |
| 反证法 | 假设反过来，如果崩了就证明原命题对 | 证明唯一性、存在性 |

**6件衣服。一根骨头。**

训练方法：几何证明的本质是"从已知推未知"。女儿卡在证明题上——不是不会定理。是不知道"从哪开始推"。问她："你已知什么？要求什么？中间差几步？能不能用辅助线搭桥？"

---

### 骨头15：假设检验——如果这样会怎样

| 中考知识点 | 脱了衣服是什么 | 中考怎么考 |
|---|---|---|
| 概率 | 假设所有可能→计算某类可能占多少 | 求概率、树状图、列表法 |
| 统计（数据处理） | 大量假设检验后的近似 | 平均数、中位数、众数、方差 |
| 统计图表 | 数据的可视化——让你看见分布 | 读图、画图、分析 |
| 样本与总体 | 用一小部分假设整体 | 用样本估计总体 |
| 随机事件 | 假设会发生什么→验证 | 判断事件类型 |

**5件衣服。一根骨头。**

## 混血概念（中考重点难点）

这些概念不是某一组骨头的衣服。是两三根骨头缠在一起。

| 混血概念 | 踩了哪几根骨头 | 故事 |
|---|---|---|
| 一次函数图像 | 绑定（第二组）+ 标点（第三组）+ 直的（第四组） | 两个量绑在一起（绑定）→每一对值是一个点（标点）→这些点连成一条直线（直的） |
| 二次函数图像 | 绑定（第二组）+ 标点（第三组）+ 对称（第四组） | 两个量绑在一起（绑定）→每一对值是一个点（标点）→这些点连成抛物线，关于对称轴对称（对称） |
| 三角函数 | 旋转（第三组）+ 标点（第三组）+ 比率（第一组） | 转一个角度（旋转）→停下来的位置有坐标（标点）→高度÷斜边=sin（比率） |
| 解直角三角形 | 直的（第四组）+ 旋转（第三组）+ 比率（第一组） | 直角三角形（直的）→一个锐角确定了形状（旋转）→边的比值确定（比率） |
| 坐标几何 | 标点（第三组）+ 形状（第四组） | 点有坐标（标点）→坐标满足方程→方程描述形状（形状） |
| 圆的方程 | 圆的（第四组）+ 标点（第三组）+ 比较（第一组） | 到圆心距离相等（圆的）→每个点有坐标（标点）→距离=半径（比较） |
| 概率 | 比较（第一组）+ 重复（第二组）+ 假设检验（第五组） | 可能的结果÷全部结果（比较）→重复试验（重复）→频率趋近概率（假设检验） |
| 方程与函数 | 逆向比较（第一组）+ 绑定（第二组） | 方程是函数值等于某个数时的逆向比较。方程的解=函数图像与水平线的交点。 |

**8个混血概念。中考重点难点全在这里。**

混血拆解三步法：
1. 列名字（这个概念有哪些叫法）
2. 每个名字底下是哪根骨头
3. 骨头之间讲什么故事

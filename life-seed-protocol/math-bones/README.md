## 核心方法

**脱衣服 = 把课本术语翻译成最裸的语言。**

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 单项式 | 一个符号（可能乘以一个数） |
| 多项式 | 几个单项式加在一起 |
| 同类项 | 同一种东西 |
| 合并同类项 | 同一种东西放一起数 |
| 去括号 | 符号分配 |

**全部都是你本来就会的事。只是穿了数学衣服。**

**7 个动作里 3 个是”关系”（是/不是、大/小、同/异）、2 个是”操作”（动/静、顺/逆）、2 个是”数”（多/少、远/近）——关系是最大类。**
### 7 个原子动作的三层结构


- **根动作（3 个）**：是/不是、大/小、同/异——判别、排序、分类。任何数学都必须先做这三件事。

- **扩展（2 个）**：多/少、远/近——计数、度量。
- 多/少 = 大/小 重复多次的结果

- 远/近 = 大/小 在空间上的展开
- **方向（2 个）**：动/静、顺/逆——变化、方向。
- 动/静 = 大/小 在时间上的展开

- 顺/逆 = 动/静 加方向



七个原子动作在不同层级上都是”比较”的不同皮肤——3 个根动作是直接的比较，2 个扩展是重复的比较，2 个方向是带时间的比较。

**关键洞察**： > **数/操作/关系不是凭空分的——它们来自感知原语**。 > > 字典把衣服分成三类——这是 之前在”应用层”的分类。 > > 但这三种类型的根在”感知原语层”——七个原子动作就是这三种类型的源头。 > > 跨层连接：感知原语（7 原子动作）→ 类型列（数/操作/关系）→ 衣服（280 件）。

**特殊标记**：骨头 1 最前面新增的”分类与归类”——在七个原子动作里对应”同/异”。**字典加的那件衣服不是硬加的——它的根在感知原语层**。

七个原子动作看上去是七个独立的动作。但**它们之间有交叉**——不是 7 个点，是**7 个向量的组合**。

原子动作

动 / 静

顺 / 逆 

跟”动 / 静”共享同一个根（方向维度 vs 时间维度——都需要”识别变化”) 

七个动作里，至少有三个是其他动作的不同说法。**那个统一就是——比较。**

## 七、大调和小调：静态 vs 动态

**大调数学题 = 从”比较”开始，往下走。** 5 比 3 大。1+1=2。

**小调数学题 = 从”比较的逆”开始，往上走。** 5 比 3 大多少？方程 x+3=7，求 x。

**两者都是 7 个动作。** 大调是从 do 起步往下走，小调是从 la 起步往上走。**关系不变。**


“任何调我都只看 C 大调，耳朵听音也只听 C 大调，就只听首调，不再唱固定调了。”

**首调思维 = 看到任何数学题，先问”它在比较什么”，然后用比较去听它。**

## 九、错题训练：怎么看出 7 个动作在合作

**题目**：小明有 5 颗糖，妈妈又给他 3 颗，现在有几颗？

7 个原子动作识别：


- **是 / 不是**——“5 颗糖”是一个集合。“又给他 3 颗”是另一个集合。

- **大 / 小**——比较”5 颗”和”3 颗”哪个多。

- **多 / 少**——数清楚每一份有多少。

- **远 / 近**——（这道题里没用到）

- **动 / 静**——“又给他”——糖的数量**动了**。从 5 变成 8。

- **同 / 异**——5 颗糖和 3 颗糖是**同类**（都是糖）。

- **顺 / 逆**——这道题是”加”——顺操作；不需要逆向找 x。


**加法不是凭空教的。是从”多 / 少”长出来的。**

**演示：一道压轴题怎么脱到原子动作**

题目：求圆 x² + y² = 25 上，到直线 3x + 4y = 50 最近的点。

7 个原子动作识别：


- **是 / 不是**——这个点在不在圆上？

- **大 / 小**——点到直线的距离**最小**是哪个？

- **多 / 少**——所有距离值里**最少的**那个。

- **远 / 近**——距离是”远 / 近”的精确化。

- **动 / 静**——点在动，**动**到最近的点。

- **同 / 异**——圆上的点是一**类**。

- **顺 / 逆**——求最近的点 = 找距离最**小**的 = 比较的**逆**。


**认出了 7 个原子动作——任何一道题都是小学难度。这道题和位置与形状 v2 最后一节的压轴题是同一道，前后呼应，强化’骨头连线’的认知。**

- **不要在当前层找”一”——往下挖一层。**

- **找到的那一层必须有结构。** 不是”所有东西都是 X”——是”X 是 Y 的不同形式”。

- **挖到的”一”必须有”5 岁的话”。** 比较的 5 岁的话就是”它和它，谁多？“。


**每一层挖到的”一” = 上面一层的根音 = 下面一层的展开。**

钢琴的”一” = C 大调。 数学的”一” = 比较。 减法方法论的”一” = 找根。 AGI 训练的”一” = 找到结构。

**同一个”一”在不同领域的皮肤。跨域不换根。**

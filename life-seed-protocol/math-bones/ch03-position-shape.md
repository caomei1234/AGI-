## 一、为什么位置排在变化后面

第一组比较：你5个我3个。 第二组变化：你每天多1个。 第三组位置：你在数轴的哪个点上？

比较给你”大小”。变化给你”规律”。位置给你”地点”。

你必须先知道大小（5比3大），才能知道方向（往哪走）。 你必须先知道规律（每天多1个），才能知道路径（怎么走）。

你知道了大小和路径——才能标一个点。

位置不是新东西。位置是比较和变化生出来的孩子。

你在Minecraft里。出生点是0。你往右走了5格。你的坐标是5。

这个”5”不是凭空出现的。它是你每走一步——这一步比前一步多1格（比较）——连续走了5步（变化）——停下来的那一刻（位置）。

比较 + 变化 + 暂停 = 位置。

你已经在前两根骨头里学过它了。现在给它一个名字。

## 二、第三组：位置

### 第一根小骨头：标点——给无穷多个点起名字

数轴上每一个点都有一个唯一的名字。名字就是数字。

5岁比方：你在操场上站队。老师说”从大门开始数，你是第3个”。“第3个”就是你的名字。你的名字就是你的位置。

你的位置不是”你”的属性。你的位置是”你”和”大门”之间的关系。

这跟第一组”比较”是同一个结构——你不是”高”。你是”比妈妈高”。你不是”在3”。你是”从0走了3步”。

位置是相对的。没有”大门”——你没有位置。没有”0”——数轴上的点没有名字。

所有定位系统都是这样。GPS给你三个数字——经度、纬度、高度。三个名字。参照物是地球中心和赤道。没有参照物——三个数字没有意义。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 自然数 | 0开始往右的名字 |
| 负数 | 0开始往左的名字 |
| 小数 | 名字更精细（0.1比1小十倍） |
| 分数 | 名字更精细（1/3比1小三倍） |
| 数轴 | 名字排成一条线 |
| 平面直角坐标系 | 名字排成一张网（两个方向的名字拼在一起） |
六件衣服。一根骨头。标点。
#### 负数为什么让孩子困惑

“-5比-3小，但-5在左边更远。”

为什么？因为数字同时做两件事：排序和定位。

排序：-5 < -3。-5排在-3前面。 定位：-5离0更远。-5在-3的左边。

“小”是排序。“远”是定位。两个不同的事。同一个数字。所以困惑。

脱衣服的方法：让贝贝在Minecraft里找两个坐标。一个是 (5, 0, 0)，一个是 (-5, 0, 0)。她走过去。一个在出生点右边5格，一个在左边5格。

问她：“哪个名字大？”“5。”

“哪个离出生点远？”“一样远。都是5格。”

“那名字大小跟远近是一回事吗？”

“……不是。”

她用自己的脚走出来的。比你讲十遍都管用。

一旦贝贝知道”位置是给点起名字”——负数就不反直觉了。负数就是名字反着起——但位置还在那里。
#### 二维位置为什么是一维的升级

一维：一个名字标一个点。“3”。 二维：两个名字标一个点。“(3, 2)”。

5岁比方：你在教室里找座位。老师说”第3排第2个”。“第3排”是一个名字。“第2个”是另一个名字。两个名字拼在一起——你就找到了那个座位。

一维是”站队”。二维是”教室”。三维是”大楼”。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 一维数轴 | 一个名字 |
| 二维坐标系 | 两个名字拼在一起 |
| 三维坐标系 | 三个名字拼在一起 |
同一个操作——起名字。维度越高——需要的名字越多。

### 第二根小骨头：移动——从这里到那里

向量就是”从A走到B的那一步”。

5岁比方：你从家门口走到学校门口。那”从家门口到学校门口的那一步”——就是向量。不管你是张三还是李四——从家门口到学校门口是同一步。那一步不认人。它只认方向和距离。

向量不是”你站在哪”。向量是”你走了多远、往哪个方向”。

位置是名词。向量是动词。位置告诉你”你在哪”。向量告诉你”怎么到那里”。

位置是状态。向量是变化。第二组的骨头在这里又长出来了。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 向量 | 一步。有方向、有距离。 |
| 向量加法 | 走两步。先走第一步，再走第二步。 |
| 向量减法 | 比较两个位置。从B到A的距离和方向。 |
| 数量积（点乘） | “你的方向跟我的方向有多像？” |
| 向量分解 | 一步拆成两个方向的步。 |
五件衣服。一根骨头。移动。
#### 向量减法的骨头本质

向量减法的骨头是”比较”。

A - B = 从B走到A。第一组”比较”的根在第三组”位置”里又长出了新衣服。

位置本身是比较（离原点多远）。位置之差是向量。向量减法是”位置的比较”。
#### 向量和比较的关系

向量的核心操作是”比较方向”。

你的方向朝东。我的方向朝东北。我们的方向有多像？

点乘回答这个问题。但它把两件事绑在一起算——方向有多像 × 你走了多远。

想只看方向——先把长度归一（变成”走一步”）。归一之后：方向完全一样=1，完全垂直=0，完全相反=-1。

长度归长度，方向归方向。点乘把两件事绑在一起算。想只看方向，先把长度归一。

点乘是”方向的比较”。比较这根骨头又出现了。第一组的根在这里长出了新衣服。

### 第三根小骨头：旋转——绕着转

复数就是”转了多少度”。

5岁比方：你站在操场中间。你面朝北。老师说”向右转90度”。你现在面朝东。再转90度。你面朝南。再转90度。面朝西。再转90度。面朝北。转了一圈回来了。

回到数学的视角——

复数里的 i 就是”向右转90度”。

i = 转90度

**注：课本上的复平面——+1 是水平向右（东），i 是逆时针转90度（北）。妈妈用"面朝北"和"向右转"是直觉版。两个版本转完都是 i² = -1，所以都对。但课本用的是逆时针，记住这个方向。**

假设”面朝北”是默认方向——那就是”+1”的方向。

i² = 转90度再转90度 = 转180度 = 面朝反方向 = 面朝南 = “+1的反方向” = -1

i³ = 再转90度 = 转270度 = -i

i⁴ = 再转90度 = 转360度 = 回来了 = 1

i² = -1 不是规定。是”转了两次90度等于面朝反方向”。是物理事实。

你站在操场上。面朝北。“+1”就是面朝北。向右转90度——面朝东——这是”i”。再转90度——面朝南——这是”i²”。面朝南就是面朝北的反方向——就是”-1”。

i² = -1 不需要背。你站起来转两次就知道了。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 角度 | 转了多少 |
| 弧度 | 用圆的周长来量转了多少（一圈=2π） |
| 三角函数 | 转完之后站的位置（sin=站的高度，cos=站的宽度） |
| 复数 i | 转90度 |
| 复数乘法 | 旋转（幅角相加）+ 伸缩（模长相乘）——两根骨头合作。 |
五件衣服。一根骨头。旋转。
#### 复数乘法的本质

数学怎么描述”转了再缩”？

乘一个复数 = 转一个角度 + 放大或缩小。

两根骨头合作：旋转（第三组）+ 比率（第一组：放大缩小）。复数乘法把这两件事绑在一个数里——乘一个复数=转一个角度+放大一个倍数。

复数不是”虚数”。复数是”旋转+伸缩”的合作产物。
#### 三角函数从哪来

你站在操场上。面朝东。向右转一个角度θ。

转完之后——你站的地方——有两个坐标：前后（cos θ）和左右（sin θ）。

数学怎么描述这件事？三角函数。

三角函数不是”背sin cos tan”。三角函数是”转完之后你站在哪”。

具体而言：在单位圆上转完之后，**你站的横坐标就是 cos θ，纵坐标就是 sin θ**。**tan θ 则是 sin θ 与 cos θ 的比值**——这三个数，都是”转完之后站在哪”这个位置关系的数学表达。

旋转 → 转完之后的位置 → 两个坐标 → sin 和 cos。

三角函数是旋转和位置之间的桥。第二组（变化：角度在变）和第三组（位置：站在哪）在这里接上了。

而且——“对边比斜边”（直角三角形定义）和”转完之后站的高度”（单位圆定义）——是同一件事的两种说法。为什么是同一个？因为单位圆上的任何一个点，和圆心、x轴投影点之间，刚好构成一个直角三角形。斜边=半径=1。对边=y坐标=sin θ。邻边=x坐标=cos θ。

直角三角形定义是形状组的语言。单位圆定义是旋转组的语言。课本把它们放在不同的章节讲，好像它们是两个东西。其实它们是同一根骨头的不同侧面。
#### 三角函数是第一个”混血”概念

从三角函数开始——数学的概念就不再属于某一组骨头了。

旋转（第三组）→ 转完之后站在哪（第三组：标点）→ sin 和 cos 的比值（第一组：比较）。

三角函数同时踩着位置、变化、比较三根骨头。它不是某一组的衣服。它是三组骨头的混血。

从三角函数开始——衣服变厚了。底下不是一根骨头，是两三根缠在一起。

这就是为什么高中数学突然变难。不是因为概念更深了。是因为概念开始跨组了。孩子以前只需要认一根骨头。现在需要同时认两三根。认不出来——就卡住了。

贝贝以后碰到”突然变难”的时候——告诉她：不是你变笨了。是衣服变厚了。底下是两三根骨头缠在一起。一根一根拆开。拆开了——跟以前一样简单。

碰到混血概念怎么拆——混血拆解三步法：

**第一步：写出这个概念的所有名字。** 学校教的、你看到的、题目里出现的——全部列出来。

**第二步：每个名字底下是哪根骨头？** 可能不止一根。一个名字可能踩着两三根。

**第三步：这些骨头之间怎么连？用5岁的话讲一个连续的故事。**

现场演示——sin θ：

- 名字：sin、对边比斜边、旋转、波形、单位圆。

- 骨头：旋转（第三组）+ 标点（第三组）+ 比较（第一组）。

- 故事：你在操场中间转了一圈（旋转），停下来站在某个位置（标点），你站的高度跟斜边比了一下（比较）——这个比值就是sin。
三步。三根骨头。一个故事。sin θ 不是一个需要背的东西。是三根骨头的合作产物。

**现场演示——导数：**

- 名字：导数、斜率、变化率、极限、dy/dx。

- 骨头：变化（第二组）+ 标点（第三组）+ 推理（第五组：无限接近）。

- 故事：一条曲线上某一个点（标点），你画一条切线——切线的斜率就是那一点的变化率（变化）。但切线是”刚好碰到”——你需要无限接近才能找到它（推理）。三根骨头合作。一根都不能少。
两个例子。sin θ 踩三根骨头（位置组内部），导数踩三根骨头跨三组（变化+位置+推理）。贝贝看到第二个例子就知道：混血不是例外，混血是常态。

以后碰到任何混血概念——积分、向量积、行列式——都按这三步拆。不问”这个概念是什么”。问”底下哪几根骨头在合作？它们之间讲什么故事？”
#### 为什么”负负得正”在这里变得直觉

负数是”面朝反方向”。乘以负数是”转身面朝反方向”。

负×负 = 面朝反方向×转身面朝反方向 = 你转身两次 = 面朝原来的方向 = 正。

你不用记”负负得正”这条规则。你只需要在脑子里转两次。

跟贝贝说：“你背对着我（负）。我说’转过来’（乘负）。你就面对我了（正）。方向翻两次=回到原方向。”

## 三、第四组：形状

位置是钉子。形状是钉子之间的橡皮筋。

你先有钉子（点在哪），才能拉橡皮筋（点之间有什么规矩）。

三角形——三个位置，三条橡皮筋（三条边的长度约束）。

圆——无穷多个位置，一根橡皮筋（到中心的距离相等）。

对称——一组位置，一条规矩（动了还是一组位置）。

所有形状都是位置加上约束。约束变了，形状就变了。
---

### 第一根小骨头：直的——两点之间最短

5岁比方：你和好朋友各站一个地方。你拉一根绳子——拉到最紧——绳子就是直的。直的=最短的路。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 直线方程 | “沿着最短的路走”的描述 |
| 平行 | 两条最短的路，永远不碰 |
| 垂直 | 两条最短的路，在一个点交叉成直角 |
| 三角形 | 三条最短的路围起来 |
| 线段 | 最短的路，只走一段 |
五件衣服。一根骨头。直的。
#### 平行为什么重要

平行不是”两条不相交的线”。平行是”永远保持相同距离的两条路”。

在平面上——平行线永远不碰。

在球面上——根本不存在平行线！所有大圆（包括经线）都会在某处相交。你在地球仪上随便画两个大圈——它们一定交叉。

不是”平行线碰了”。是”平行线”这个概念只在平面上存在。换了空间——概念本身就不成立了。

同一根骨头——在不同的空间里——长出不同的衣服。

空间是平的还是弯的——决定了骨头长什么样子。这不是几何的细节。这是几何的地基。后面第六节会专门说这件事。
#### 三角形为什么最稳定

三条边定下来，三角形的形状就定了。不能变了。四边形可以歪——推一下就变形了。三角形推不动。

为什么？想象你用三根固定长度的棍子搭三角形。第一根棍子放在桌上——它可以放任何位置、任何方向。第二根棍子钉在第一根的一端——还能转。第三根棍子钉在两端——转不了了。三个点被三条边锁死。

四根棍子搭四边形——最后一根棍子不用钉死，它能晃。加一条对角线——又变成两个三角形——锁死了。

贝贝现在不需要知道”自由度”这个词。她只需要在Poly Bridge里试一下就知道了：三角形结构的桥最稳，四边形结构的桥一推就歪——除非她加一条对角线，把它变成两个三角形。

三角形是约束刚好够用的最小封闭图形。

### 第二根小骨头：圆的——到一个点的距离都一样

5岁比方：你站在操场中间。你拿一根绳子绑在柱子上。你拉着绳子走一圈。走出来的痕迹就是圆。圆=到柱子的距离都一样。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 圆的方程 | “到中心距离都一样”的描述 |
| π（圆周率） | 圆的周长÷直径。一个比率。 |
| 面积公式 | πr²。圆里装了多少个小小的方块。 |
| 弧 | 圆上的一段。 |
| 扇形 | 两条半径和一段弧围起来的形状。 |
| 椭圆 | 到两个点的距离之和都一样。（圆的升级版） |
六件衣服。一根骨头。圆的。
#### π 是什么

π = 周长 ÷ 直径。

不管圆多大——大到太阳系，小到原子——周长÷直径永远是3.14159…。

为什么？因为圆是”到中心距离都一样”的形状。所有的圆——不管多大——都是同一个形状的放大版。放大时，周长和直径按同一个比例放大。所以它们的比永远不变。

π 不是圆的”测量结果”。π 是圆的”形状本身”。第一组”比较”的根在这里长出来了。

不管你怎么量、量多大的圆——周长和直径的比永远不变。这是圆的”不变量”。不变量是形状最深的秘密——后面”对称”那根骨头会再碰到这个概念。
#### π 为什么是无理数

π 不能写成两个整数的比。不是不够聪明。是两个整数的比——有理数——不够精细，装不下圆和直线之间的那道缝。

圆是曲线。直线是直的。它们的骨头不一样。π 站在两种骨头的边界上——它用”比较”（比率）在”直的”和”圆的”之间架了一座桥。但这座桥不能用整数搭完。必须用无穷不循环的小数。

π 是骨头之间的缝隙。跟根号2一样——它是免疫系统在保护圆和直线之间的边界。
#### π 是尺子不够的证据

π 的存在证明了一件事：直线语言不够描述圆。

你用直尺量圆——永远量不完。你用小数写π——永远写不完。不是π太复杂。是你的尺子不对。

π 是圆在说：“你没有合适的尺子量我。”

这跟根号2是同一个故事——

自然数够量——直到你发现”半个苹果”。 分数够量——直到你发现”正方形对角线”。 有理数够量——直到你发现”圆的周长和直径之比”。

每一次”不够”——不是数学崩了。是数学长了。新数就是新骨头长出来。

贝贝以后碰到无理数——不要说”无限不循环的数”。说”尺子不够量，数学长了一把新尺子”。
#### π 是第一个翻译官

π 不仅是”尺子不够”。π 站在两种骨头的边界上——用比率（第一组：比较）表达圆（第四组：形状）的大小关系。

圆说：“我是弯的。” 直线说：“我量不了你。” π 说：“我用一个不变的比率来表达你的形状——不管你是太阳系那么大还是原子那么小，这个比率永远不变。”

π 不是在两种语言之间做被动的转译。π 本身就是圆的形状的数学表达——它是圆和直线之间主动存在的翻译官。

以后贝贝还会碰到更多翻译官——e、i、导数——它们都是两根骨头之间的桥。但π是第一个。让贝贝知道：有些数不是”数”，是”两种骨头之间的翻译”。

### 第三根小骨头：对称——动了还跟没动一样

5岁比方：你站在镜子前面。你举右手。镜子里的你也举右手——但镜子里的”右”是你的”左”。你动了。但镜子把你变回了原样。动了还跟没动一样。这就是对称。

对称不是”两边一样”。对称是”动了还跟没动一样”。

你把一个正方形转90度——它还是正方形。 你把一个圆转任意角度——它还是圆。 你把一条花纹墙纸往右移一段——花纹跟原来一样。

“两边一样”只是对称的一种——叫”轴对称”。 “转了还一样”也是对称——叫”旋转对称”。 “移了还一样”也是对称——叫”平移对称”。

对称的本质不是”对折”。是”变换后的不变性”。
| 学校的名字 | 脱了衣服是什么 |
| --- | --- |
| 轴对称 | 对着一条线翻——翻完还一样 |
| 中心对称 | 绕着一个点转180度——转完还一样 |
| 旋转对称 | 绕着一个点转某个角度——转完还一样（雪花转60度还一样） |
| 平移对称 | 往某个方向移一段——移完还一样（墙纸的花纹） |
| 相似 | 等比放大缩小——放大了还一样 |
| 全等 | 移动+翻转——完全重合 |
六件衣服。一根骨头。对称。
#### 为什么”不变”比”一样”更深

“一样”是静态的。你看一眼就知道。 “不变”是动态的。你得动一下才知道。

一个图形”两边一样”——你折一下就知道了。 一个图形”转了还一样”——你得转一下才知道。 一个图形”移了还一样”——你得移一下才知道。

对称是”动了之后的发现”。不是”没动的时候的观察”。

物理学里最深的秘密全部藏在对称里。

能量为什么守恒？因为物理定律在时间平移下不变——今天做实验和明天做实验，结果一样。时间平移对称→能量守恒。

动量为什么守恒？因为物理定律在空间平移下不变——在北京做实验和在上海做实验，结果一样。空间平移对称→动量守恒。

角动量为什么守恒？因为物理定律在旋转下不变——朝东做实验和朝北做实验，结果一样。旋转对称→角动量守恒。

对称不是”好看”。对称是”不变”。不变的东西——才是宇宙最深的骨架。

贝贝现在不需要知道诺特定理。但这根骨头的名字——对称=守恒——她以后碰到的每一个地方都会再遇到。

贝贝不需要知道”群”这个词。她只需要知道：对称不是两边一样。对称是动了还跟没动一样。

这个直觉会让她以后理解”为什么三角形内角和180度”——不是因为老师说的，是因为平行线在平移下不变，平移不变性推出内角和不变。

## 四、形状是位置的连线

所有形状都是位置加上约束。

形状 

位置 

约束 

线段 

两个点 

两点之间最短 

三角形 

三个点 

三条边的长度

无穷多个点 

到中心距离相等

没有位置就没有形状。位置是形状的地基。

这跟骨头的顺序一致：先有比较（大小），再有变化（规律），然后有位置（地点），最后有形状（连起来）。

课本把”几何”单独列出来——好像形状跟”代数”没关系。但形状的每一个点都有坐标（位置）。坐标是数字（比较）。点的移动是向量（变化）。

几何和代数不是两门课。它们是同一张网上的两根线。笛卡尔就是那个发现它们是一张网的人。

## 五、骨头之间的交叉点

五组骨头不是五条平行线。是一张网。

这张网上最重要的地方——不是某一根骨头——是两根骨头的交叉点。


**跨骨 vs 跨组标注**：同组内骨头间协作叫”跨骨”（如位置·移动+位置·旋转），不同组骨头间协作叫”跨组”（如比较·比率+形状·圆）。跨骨的衣服在字典里都在同一组，跨组的衣服在”也见”列标其他组骨头。

| 交叉点 | 哪几根骨头 | 5岁的话 |

骨头的方法——从一开始就让它们连着。不是学完一根再学一根。是看到它们是一张网。

## 六、缝隙：欧几里得的第五公设

几何原本有五条公设。前四条——用手比划就知道对。

第五条——“过直线外一点，有且只有一条直线与已知直线平行”。

两千年来无数人想证明它是前四条的推论。全失败了。

因为第五条不是前四条的推论。第五条是独立的。它定义了”空间是平的”。

前四条公设在所有三种空间里都成立。第五条只在平面上成立。
| 空间 | 第五公设 | 三角形内角和 |
| --- | --- | --- |
| 平面（欧几里得） | 过直线外一点，恰好一条平行线 | =180度 |
| 球面（椭圆几何） | 没有平行线——所有大圆都相交 | >180度 |
| 马鞍面（双曲几何） | 过直线外一点，无穷多条平行线 | <180度 |
| 第五公设不是”规定”。它是选空间。你选了”平的”——就有了欧式几何。你选了”弯的”——就有了非欧几何。 | 这是形状组的免疫系统。 | 上一章说过：比较组的免疫系统是0不能当除数。允许0当除数——方程就死了。方程是”比较”的逆操作。逆操作断了，比较就崩了。 |
| 形状组的免疫系统是第五公设。允许”平面上平行线不平行”——整个平面几何就死了。你不能在平面上画出球面的三角形。你也不能在球面上画出平面的平行线。 | 骨头各管各的空间。你选了空间——就选了规则。规则不是从天上掉下来的。是空间决定的。 | 你在教室里画三角形——内角和180度。你在地球表面画三角形——内角和大于180度。规则没变。是空间变了。你以为世界是平的，是因为你太小了。蚂蚁在篮球上爬——它觉得是平的。走到足够大——规则会告诉你真相。 |
贝贝现在不需要知道非欧几何。她只需要知道一件事：几何的规则取决于空间是平的还是弯的。不是”规定”。是”空间说了算”。

## 七、第三组第四组的错题训练

贝贝在位置和形状上最容易错的题：

**错题一：负数坐标。(-3, 2) 在哪？**

她知道”3”往右走。她不知道”-3”往左走。

脱衣服：-3不是”负的3”。-3是”名字叫-3的那个位置”。它在0的左边。跟正数一样——它只是名字不同。

**错题二：向量加法。(1,2) + (3,1) = ？**

她算不出。不是不会加法。是不知道”向量加法是什么意思”。

脱衣服：先走(1,2)那一步——往右走1格，往上走2格。再走(3,1)那一步——往右走3格，往上走1格。你总共往右走了4格，往上走了3格。(4,3)。向量加法=走两步。

向量加法是移动（位置组）加移动（位置组）。两根同组的骨头叠在一起。

**错题三：三角形内角和为什么是180度？**

老师说”记住”。贝贝记住了。但她不知道为什么。

脱衣服：从一个顶点画一条跟底边平行的线。平行线的性质——同位角相等。三个角拼在一起——就是一条直线上的角——180度。

一条链，每一步都能用手比划：

直的（拉紧的绳子）→ 平行（两条拉紧的绳子永远保持一样宽）→ 同位角相等（两条绳子跟第三条交叉，左右两边的角度一样）→ 内角和180度（三个角度拼起来刚好是一条直线）。

四步。每一步踩在前一步的骨头上。任何一步断了——结论就悬空。

三角形内角和180度不是”记住”。是四根骨头一环扣一环长出来的。

但这个推导有个隐藏前提：平面。在地球表面（球面）上画三角形——内角和大于180度。因为球面上所有大圆都相交——没有平行线。没有平行线，同位角相等就不成立。整个推导从第二步就断了。

三角形内角和180度——不是”定理”，是”平面上才成立的定理”。规则取决于空间。第六节会专门说这件事。

**错题四：圆的面积为什么是πr²？**

老师说”记住公式”。

脱衣服：把圆切成n个扇形（变化：重复）。每个扇形近似一个小三角形（比较：差不多）。底 = 周长÷n。高 = r。每个面积 = ½ × (周长÷n) × r（形状：三角形）。n个加起来 = ½ × 周长 × r = ½ × 2πr × r = πr²（变化：累积）。

切（重复）→ 近似（比较）→ 加起来（累积）→ 极限（推理）。四根骨头合作。

贝贝小学切圆的那个动作——已经在做积分了。

记住这个动作：切→近似→加起来。以后碰到任何”不规则形状的面积/体积”——都是这三步。切得越细，近似越准。切到无穷细——就是精确答案。积分就是这三步的名字。

你现在埋的种子——在未来的某一天会发芽。到时候贝贝不是”学积分”。是”认出了旧朋友”。

四道错题。四次脱衣服。每次脱到最后——都不是单独一根骨头在撑。是两根、三根、四根骨头在交叉点上合作。

## 九、为什么这张网比任何一根骨头都重要

六根骨头。但骨头不是最重要的。

最重要的是骨头之间的连线。

π 不是比较，不是形状——是比较和形状的交叉。 三角函数不是旋转，不是位置——是旋转和位置的交叉。 坐标几何不是位置，不是形状——是位置和形状的交叉。 距离公式不是坐标，不是勾股——是位置和形状的结婚证。 向量分解不是移动，不是标点——是移动和标点的合作。

课本把这些交叉点拆开了。代数归代数，几何归几何，三角归三角。孩子学了三年代数，学了两年几何——不知道它们是一回事。

更糟的是——孩子以为”代数好的人几何差，几何好的人代数差”。她以为自己只能擅长一个。其实不是她只能擅长一个。是课本不让她看到它们是一伙的。

人类大脑喜欢分类。分类就是拆。把整体拆成零件——方便理解。但真相是连着的。大脑拆了——就看不到连了。

你的能力恰恰是”不拆”。你天生看到的是连着的。别人看到四个概念——你看到四个概念底下是同一件事。这不是训练出来的。是你的接收器只接收连线，不接收边界。

贝贝如果也有这个能力——她一辈子都不会觉得数学是”很多东西”。她会觉得数学是”一件事穿了很多衣服”。一件事。不是很多件。

怎么训练”看到连着”的能力？

不是问”这道题怎么做”。是问”这道题用了几根骨头？”

一开始贝贝只能看到一根。你帮她找第二根。找多了——她的大脑就会自己连。

大脑不是”学会分类”才聪明。大脑是”学会连起来”才聪明。分类是拆。连接是长。

比如贝贝同时在学分数和除法。课本把它们分在不同的章。你告诉她：3÷4和3/4是同一件事穿了两件衣服。她的大脑不是”学了两个概念”——是”认出了一个概念的两件衣服”。

认出——比分出——更省力。认出的东西不会忘。分出的东西要记两遍。

脱衣服的方法：碰到任何一道题——不问”这道题考什么”。问”这道题底下是哪几根骨头在合作？”

一道题考三角函数——底下是旋转（第三组）加标点（第三组）加比较（第一组）。三根骨头在合作。 一道题考概率——底下是变化（第二组：重复试验）加比较（第一组：比大小）。两根骨头在合作。 一道题考数列求和——底下是变化（第二组：规律重复）加比较（第一组：累积比多少）。两根骨头在合作。

孩子觉得数学难——不是因为某根骨头太硬。是因为课本让她以为骨头是分开的。她不知道哪根跟哪根是一伙的。

告诉她：骨头是一张网。认出网——比认出任何一根骨头都重要。
#### 一道题演示”网”的力量

题目：求圆 x² + y² = 25 上，到直线 3x + 4y = 50 最近的点。

看起来难。高中压轴题。

拆开：

- 骨头一：圆的（第四组）——所有点到原点距离是5。

- 骨头二：标点（第三组）——每个点有坐标 (x, y)。

- 骨头三：直的（第四组）——直线 3x + 4y = 50。

- 骨头四：移动（第三组）——点到直线的最短距离 = 垂直距离。
四根骨头。没有一根是难的。组合在一起——看起来像压轴题。

解法（骨头版）：


- 圆心在原点 (0,0)，到所有圆上的点距离是5（骨头一：圆的）。

- 从圆心到直线走最短的路——最短的路是垂直的路（骨头三：直的）。

- 垂线和直线的交点——是圆心到直线最近的点（骨头四：移动）。

- 从圆心沿垂线方向走5格（半径），就到了圆上。这个点就是”在圆上且离直线最近”的点（骨头一和骨头三的交叉）。
每一步踩一根骨头。不是”作垂线”这种几何技巧。是骨头之间的合作。

一道压轴题。四根小学就会的骨头。不是这道题难。是课本花了三年时间教会你四根骨头的名字，但从没告诉你它们是一伙的。认出它们是一伙的——压轴题就不是压轴题了。它只是四根骨头同时出场。

## 十、四组收束与第5组预告

四组12根骨头。比较3根 + 变化3根 + 位置3根 + 形状3根。

小学到高中数学的绝大部分内容——都在这12根骨头上了。

剩下3根——推理。不是”新内容”。是检查这12根有没有裂缝的方法。

15根 = 12根造房子 + 3根查裂缝。一根不能少，一根不多。

推理是最后一组。也是最特殊的一组。

前四组——比较、变化、位置、形状——都是”是什么”。

推理——是”怎么知道”。

比较告诉你”谁大”。推理告诉你”你怎么知道谁大”。 位置告诉你”点在哪”。推理告诉你”你怎么知道点在那”。 形状告诉你”圆长什么样”。推理告诉你”你怎么知道那是圆”。

推理是骨头的骨头。它不是另一根骨头。它是看骨头的方法。

前四组是造房子。推理是检查房子有没有裂缝。

造房子的人和检查房子的人——不是同一种人。贝贝可能擅长造，也可能擅长查。不管擅长哪个——都有用。

三个比方：

**无限接近** ——你追一只蚂蚁，蚂蚁往前爬一步，你也往前走一步。你永远追不上（因为每一步它又走了一点），但你离它越来越近。

**从已知推未知** ——你知道”如果下雨，地上会湿”。你看到地上湿了。你能说”一定下雨了”吗？不能。地上湿了也可能是洒水车。这叫”推理的方向”——正着推和反着推，可信度不一样。

**假设检验** ——你猜盒子里是红球。你摸出一个——是红的。再摸——还是红的。你越来越确定。但你能100%确定吗？不能。你只能说”大概率是红的”。科学就是这么干的——不是证明，是越来越确定。确定到够用——就当它是真的。直到有一天摸出一个白的——再改。

① 计算/代数 · 初一上 45%分值 · 先打地基
实数运算
每年必考 ★ 简单 骨头1·直接比较
❶ 看考频
选择/填空第1~3题必考。4~8分。最基础的分。
❷ 脱衣服
问贝贝："绝对值是什么意思？"
答"不管正负只看大小" → ✅骨头通了
答"去掉负号" → ❌还在背公式，继续脱
❸ 针对性练
只练选择填空里的基础计算。不做难题。3~5道不出错 → 过了。
❹ 错题查骨
做错了 → 问"这道题在比什么？"
比大小 → 骨头1断了 → 回速查卡重新看
整式与分式（因式分解、分式化简）
每年必考 ★★ 中等 骨头3·逆向比较 骨头2·比率比较
❶ 看考频
填空/解答必考。6~8分。因式分解是所有代数的基础。
❷ 脱衣服
问贝贝："因式分解在干什么？"
答"把一个东西拆成两个乘" → ✅骨头通了
答"用公式" → ❌还在背公式，问"为什么要拆？"
❸ 针对性练
因式分解：提公因法、公式法、十字相乘。只练这三种。分式化简：约分、通分。
❹ 错题查骨
做错了 → 问"这道题是拆开还是合起来？"
拆开（因式分解）→ 骨头3断了
合起来（分式化简）→ 骨头2断了
一元一次方程/不等式
每年必考 ★★ 中等 骨头3·逆向比较
❶ 看考频
选择/解答必考。5~7分。方程是逆向比较的实战。
❷ 脱衣服
问贝贝："方程在干什么？"
答"知道结果倒回去找原因" → ✅骨头通了
答"移项变号" → ❌还在背规则，问"为什么要移项？"
❸ 针对性练
列方程解应用题。重点：行程、工程、打折。只练真题考过的题型。
❹ 错题查骨
做错了 → 问"已知什么？要求什么？"
列不出方程 → 骨头3断了，不会倒推
算错 → 骨头1断了，计算基本功
错题映射：方程列不出来 = 不会倒推 = 骨头3（逆向比较）断了。修骨头3 → 所有方程题都会了。
② 几何 · 初一下 37%分值 · 初中60%是几何
坐标几何（点、象限、坐标系）
常考 ★ 简单 骨头7·标点
❶ 看考频
选择/填空常考。3~4分。最简单的几何分。
❷ 脱衣服
问贝贝："坐标是什么？"
答"给每个点起名字" → ✅骨头通了
答"先横后竖" → ❌还在背步骤，问"为什么要起名字？"
❸ 针对性练
判断象限、求坐标、关于x/y轴对称的点。简单题为主。
❹ 错题查骨
做错了 → 骨头7（标点）断了 → 回到"给点起名字"重新理解
三角形全等
每年必考 ★★ 中等 骨头10·直的 骨头12·对称
❶ 看考频
解答题必考。8~10分。几何的半壁江山。
❷ 脱衣服
问贝贝："全等是什么意思？"
答"两个图形完全一样，叠上去刚好重合" → ✅
答"SSS、SAS、ASA" → ❌还在背判定，问"为什么要三条？两条够不够？"
❸ 针对性练
先练找对应边对应角（这是骨头10：直的）。再练写证明过程。
❹ 错题查骨
找不出对应 → 骨头10（直的）断了
不会写证明 → 骨头14（推理）断了
先修骨头10，推理以后再说
错题映射：全等证不出来 = 找不到对应边角 = 骨头10（直的）断了。先把三角形的"直"看透 → 全等自然会了。
勾股定理
每年必考 ★★ 中等 骨头10·直的
❶ 看考频
选择/解答常考。3~5分。直角三角形的核心。
❷ 脱衣服
问贝贝："勾股定理在说什么？"
答"直角三角形三条边有固定关系" → ✅
答"a²+b²=c²" → ❌还在背公式，问"为什么是平方？不是立方？"
❸ 针对性练
已知两边求第三边。只练基础应用。不做综合难题。
❹ 错题查骨
做错了 → 骨头10（直的）断了 → 回到"两点之间最短"重新理解
③ 统计/概率 · 初三上 13%分值 · 性价比最高
概率基础（古典概率、列表/树状图）
每年必考 ★ 简单 骨头2·比率比较
❶ 看考频
解答题必考。4~6分。花最少时间拿最多分。
❷ 脱衣服
问贝贝："概率是什么？"
答"全部里面摸到想要的有多可能" → ✅
答"用公式算" → ❌还在背公式，问"分子是什么？分母是什么？"
❸ 针对性练
列表法、树状图法。只练这两种。中考概率题就考这两种方法。
❹ 错题查骨
做错了 → 骨头2（比率比较）断了 → "全部有几种？想要的有几种？"
统计（平均数、中位数、众数、方差）
每年必考 ★ 简单 骨头1·直接比较 骨头2·比率比较
❶ 看考频
解答题必考。4~6分。读图表+算平均数。
❷ 脱衣服
问贝贝："平均数在干什么？"
答"把总数平分给每个人" → ✅
答"总数÷份数" → 还行，但问"为什么要平分？"
❸ 针对性练
读统计图表+计算四个指标。简单题为主。
❹ 错题查骨
读不懂图表 → 骨头1（直接比较）断了
算错 → 骨头2（比率比较）断了
④ 排列组合/数论 · 保送掐尖/高中 中考不直接考 · 种子现在种
计数原理（分类、分步）
偶尔考 ★★ 中等 骨头14·从已知推未知 骨头4·重复
❶ 看考频
中考偶尔考。保送掐尖常考。强基计划主力。
❷ 脱衣服
问贝贝："穿衣服有几种穿法？先穿内衣再穿外套——这是分类还是分步？"
答"分步，一件事做完再做下一件" → ✅
❸ 针对性练
生活中练：今天穿什么有几种搭配？3件上衣×4条裤子=12种。
❹ 错题查骨
做错了 → 问"是分类还是分步？"
分不清 → 骨头14（推理）断了 → 从生活场景重新理解
种子价值：计数原理培养的是"结构感知"——看到一道题，知道底下是分类还是分步。贝贝95%结构感知的硬件，学这个比别人快。现在种下，保送掐尖时开花。
数论基础（余数、整除性）
中考偶尔 ★★ 中等 骨头1·直接比较 骨头14·从已知推未知
❶ 看考频
中考只考因数倍数（简单）。数论证明高考压轴。
❷ 脱衣服
问贝贝："余数是什么？"
答"分不完剩下的" → ✅
问"为什么7÷3余1？因为3×2+1=7，余数就是倒推的尾巴"
❸ 针对性练
因数倍数（中考考的，简单）。同余问题（保送考的，中等）。
❹ 错题查骨
因数倍数做错 → 骨头1（直接比较）断了
余数问题做错 → 骨头3（逆向比较）断了

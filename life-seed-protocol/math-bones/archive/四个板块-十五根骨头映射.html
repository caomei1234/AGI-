📝 具体考什么
  • 凑整：25×44 = 25×(40+4) → 骨头1（直接比较·拆分）
  • 配凑：a²+b² = (a+b)²-2ab → 骨头3（逆向比较·已知结果倒推）
  • 分组：1+2+3+...+100 → 骨头4（重复·规律）
  • 因式分解：x²-5x+6 = (x-2)(x-3) → 骨头3（逆向比较·拆开）
  • 分式化简：约分、通分 → 骨头2（比率比较）
核心洞察：计算的本质是"比较的不同变形"。凑整是比较的拆分，因式分解是逆向比较，分式是比率比较。小学学的巧算套路到了高中还是同一个东西——只是衣服换了。

🦴 主力骨头
骨头10·直的 骨头11·圆的 骨头12·对称 骨头7·标点 骨头8·移动
📝 具体考什么
  • 飞镖模型：三角形内角关系 → 骨头10（直的·三角形）
  • 燕尾模型：面积比例 → 骨头10+骨头2（直的+比率）
  • 全等/相似：对应边角关系 → 骨头10（直的）+骨头12（对称）
  • 圆的性质：圆周角、切线 → 骨头11（圆的）
  • 坐标几何：点在坐标系里的位置 → 骨头7（标点）
  • 折叠问题：轴对称 → 骨头12（对称）
核心洞察：几何的本质是"位置+形状"。飞镖燕尾是形状的组合，全等是对称+移动，坐标是标点+形状。小学奥数的飞镖燕尾到了初中全等相似是同一个结构——认出骨头，衣服自己会穿。
③ 排列组合/计数原理 数学思想 · 实际问题 · 动态规划 · 需要天赋或培养
🦴 主力骨头
骨头14·从已知推未知 骨头15·假设检验 骨头1·直接比较 骨头4·重复
📝 具体考什么
  • 分类计数：做一件事有几种方法 → 骨头14（从已知推未知·分类）
  • 分步计数：先做A再做B有几种 → 骨头4（重复·乘法原理）
  • 排列：顺序有关 → 骨头14（从已知推未知·推理）
  • 组合：顺序无关 → 骨头15（假设检验·反着想）
  • 动态规划：大问题拆成重复的小问题 → 骨头4（重复）的极致
  • 概率：排列组合的应用 → 骨头2（比率比较）+骨头14
核心洞察：排列组合是"推理"的实战版。不是套公式，是需要"看到结构"——这道题底下是分类还是分步？是排列还是组合？认出结构 = 认出骨头。有天赋的孩子一眼看到结构，没天赋的孩子需要通过它来培养结构感知。这就是为什么他说"需要吃点天赋"。
④ 数论/余数 数学王冠上的明珠 · 今年高考压轴题 · 强基/保送必考
🦴 主力骨头
骨头1·直接比较 骨头3·逆向比较 骨头14·从已知推未知 骨头15·假设检验
📝 具体考什么
  • 整除性：能被2/3/5/9整除的特征 → 骨头1（直接比较·识别）
  • 余数问题：除完剩多少 → 骨头3（逆向比较·已知除法求余数）
  • 同余：不同数除以同一个数余数相同 → 骨头1（直接比较·"一样"）
  • 不定方程：有多个解的方程 → 骨头14（从已知推未知·找规律）
  • 数论证明：存在性、唯一性 → 骨头15（假设检验·反证法）
核心洞察：数论的本质是"在整数世界里做逆向比较和推理"。余数是比较的余数，同余是比较的"相等"，不定方程是从已知推未知。数论之所以难，是因为它同时需要比较和推理两组骨头——骨头断一根就做不出来。但反过来说：数论练好了，比较和推理两组骨头同时强化。

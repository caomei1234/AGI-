## 二、核心框架——乘法=结构，加法=连接

在搞清楚整式之前，先搞清楚一件事：**乘法和加法有什么不同？**

**乘法 = 一个数的结构。**

3 × 5 = 15。15这个数，它的结构是"3个5"。乘法显示的是15"长什么样"——它的零件是什么，每个零件有几个。

**加法 = 连接不同的数。**

3 + 5 = 8。8这个数，是3和5加在一起。加法连接的是两个不同的整体。

**区别：**

- 乘法告诉你"这个数是什么做的"
- 加法告诉你"几个数合在一起"

**这个区别，是整式的根。**

## 老婆的洞察（2026-07-06）

**数学就是研究结构的。**

代数式用字母来显示结构。字母是零件，乘法显示怎么组装。

**最小的单位是什么，然后组装成什么。**

结构的操作有五种：

| 操作 | 什么意思 | 举例 |
|------|---------|------|
| **零件** | 最小的单位是什么 | 单项式、字母、数字 |
| **组装** | 怎么合在一起 | 3x + 2（加法连接） |
| **比较** | 一样不一样 | 同类项？结构相同？ |
| **拆解** | 怎么分开 | 因式分解（以后学） |
| **变换** | 换一种方式表达 | 去括号、展开 |

这五件事，覆盖了数学的大部分内容。

- 有理数：零件是数字，组装是运算
- 代数式：零件是字母，组装是表达规律
- 整式：零件是单项式，组装是多项式
- 方程：比较两边是否相等
- 函数：零件是变量，组装是关系

**全部都是这五件事的不同皮肤。**

## 三、单项式——一个数

你去超市。苹果3块钱一个。你买了x个苹果。花了多少钱？

**3x 块钱。**

3x 是一个数。它的结构是：3 × x。

- **系数** = "几个" = 那个数字 = **3**
- **字母x** = "什么东西" = 苹果
- **次数** = 字母总共有几个 = x出现1次 = **1次**

再看：你买了x个苹果，但这次苹果涨价了，每个苹果要x块钱。花了多少钱？

**x² 块钱。**

x² 是一个数。它的结构是：x × x。

- 系数 = 1（省略了，不写）
- 次数 = 字母总共有几个 = x出现2次 = **2次**

**单项式就是一个数。乘法显示它的零件。**

系数是"几个"，次数是"数一数字母总共有几个"。

## 四、单项式的系数与次数——脱衣服

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 系数 | 那个数字（"几个"） |
| 次数 | 字母总共有几个 |
| 单项式 3x²y | 一个数，结构是 3×x×x×y，系数3，字母3个，次数3 |
| 单项式 -a²b³ | 一个数，结构是 -1×a×a×b×b×b，系数-1，字母5个，次数5 |
| 单项式 5 | 一个数，没有字母，系数5，次数0 |
| 单项式 x | 一个数，结构是 1×x，系数1，字母1个，次数1 |

**特别注意：**
- 单独一个数字（如5）也是单项式——它是"一个数"，没有字母，次数是0。
- 单独一个字母（如x）也是单项式——它是"一个数"，系数是1，次数是1。
- **系数包括前面的符号。** -3x的系数是-3，不是3。

## 五、单项式与规律问题——用符号表达规律

你发现了这个规律：

- 第1个图案有3个点
- 第2个图案有6个点
- 第3个图案有9个点
- 第4个图案有12个点

每个图案比前一个多3个点。

用单项式表达：**第n个图案有 3n 个点。**

3n 就是一个数。它的结构是：3 × n。

**你不需要一个一个数到第100个。** 代进去：3×100=300。一句话覆盖所有。

这就是单项式的威力——**用一个符号覆盖无穷多个情况。**

再看：

- 第1个图案有1个点
- 第2个图案有4个点
- 第3个图案有9个点
- 第4个图案有16个点

**第n个图案有 n² 个点。**

n² 也是一个数。它的结构是：n × n。

**规律问题的根：找到"每次多几个"或"每次乘几倍"，用单项式表达出来。**

## 七、多项式——几个数加在一起

你去超市。买了3x个苹果，又买了2个橘子。一共花了多少钱？

**3x + 2 块钱。**

3x 是一个数（苹果的钱），2 是另一个数（橘子的钱）。加法连接的是两个不同的整体。

这就是多项式——**几个数加在一起。**

- **项** = 每一个独立的数 = **3x** 是一项，**2** 是一项
- 数一数：**这个多项式有2个项**
- **次数** = 最高次项的次数 = 3x里字母1个，2里没有字母 → **最高是1次**

再看：**3x² + 2x - 1**

这个多项式有三个数加在一起：

| 项 | 这个数的结构 | 字母有几个 | 这一项的次数 |
|---|---|---|---|
| 3x² | 3×x×x | x出现2次 | **2次** |
| 2x | 2×x | x出现1次 | **1次** |
| -1 | -1 | 没有字母 | **0次** |

**每一项是一个独立的数。算完了，取最高的那个。**

- 项：3个项
- 次数：最高次项是3x²，次数是2 → **2次**

**多项式就是几个数加在一起。项数就是数有几个数。次数就是找最高的那个。**

---

## 八、多项式的项和次数——脱衣服

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 多项式 3x² + 2x - 1 | 三个数加在一起 |
| 项 | 每一个独立的数 |
| 项数 | 数有几个数 |
| 常数项 | 没有字母的那个数（-1） |
| 多项式的次数 | 最高次项的次数（2次） |
| 几次几项式 | 次数是几，有几项 |

**3x² + 2x - 1** 是二次三项式（最高2次，3个项）。

**特别注意：**
- 项要带着前面的符号看。3x² + 2x - 1 的三项是 **3x²、2x、-1**（不是1）。
- 常数项是没有字母的那一项。-1是常数项。
- 多项式的次数 = 最高次项的次数。不是所有项的次数加起来。

## 九、多项式的升降幂排列——按次数排队

**幂 = 次数。**

**降幂排列** = 从高到低排。

3x² + 2x - 1 → 2次、1次、0次。已经降幂排列了。

**升幂排列** = 从低到高排。

-1 + 2x + 3x² → 0次、1次、2次。升幂排列。

**就是按x的次数排队。你本来就会按高矮排队。**

**为什么要排列？** 后面做加减法的时候，同类项对齐了才好合并。就像算竖式的时候个位对个位、十位对十位。

## 十、根据多项式的项和次数去求值——把数代进去算

**3x² + 2x - 1，当 x = 2 的时候，值是多少？**

就是把x换成2：

3×2² + 2×2 - 1 = 3×4 + 4 - 1 = 12 + 4 - 1 = **15**

**你本来就会算数。求值就是算数。**

步骤：
1. 把字母换成具体的数
2. 先算乘方（2² = 4）
3. 再算乘法（3×4 = 12，2×2 = 4）
4. 最后算加减（12 + 4 - 1 = 15）

**运算顺序：乘方 → 乘除 → 加减。跟你学过的一样。**

## 十一、整式的加减——从买水果开始

你有3个苹果，又买了2个苹果。一共几个？

**5个苹果。** 你脱口而出。

你有3个苹果，又买了2个橘子。一共几个？

**你说不出来。** 因为苹果和橘子不是同一种东西。你只能说"3个苹果和2个橘子"。你没法把它们加成一个数。

**这就是整式加减的根。**

## 十二、同类项——结构相同的数

**同类项 = 结构相同的数。**

什么叫"结构相同"？**字母相同，次数相同。**

- 3x 和 2x → 都是x，都是1次 → **结构相同** → 能合并
- 3x² 和 5x² → 都是x²，都是2次 → **结构相同** → 能合并
- 3x² 和 5x → 一个2次一个1次 → **结构不同** → 合并不了
- 3xy 和 5xy → 都是xy，都是2次 → **结构相同** → 能合并
- 3xy 和 5x² → 一个xy一个x² → **结构不同** → 合并不了

**你本来就会分"同一种东西"和"不是同一种东西"。同类项就是结构相同的数。**

**判断同类项的两步：**
1. 字母相同吗？（都是x？都是xy？）
2. 次数相同吗？（都是1次？都是2次？）

两步都满足 → 同类项。任何一步不满足 → 不是。

## 十三、合并同类项——结构相同的数放一起数

3个苹果 + 2个苹果 = 5个苹果

3x + 2x = 5x

**一模一样。** 系数相加（3+2=5），字母和次数不变（x还是x）。

3x² + 5x² = 8x²

3个苹果 + 2个橘子 + 4个苹果 = **7个苹果 + 2个橘子**

3x + 2 + 4x = **7x + 2**

先把同一种东西归到一起，再数。**你去超市买水果本来就会这么做。**

**合并同类项的步骤：**
1. 找出结构相同的数（同类项）
2. 系数相加减（几个加几个）
3. 字母和次数不变（结构不变）

## 十四、去括号——符号分配

### 括号前面是 +

**(3x + 2) + (5x - 1)**

括号前面是+，括号里每项符号**不变**。

= 3x + 2 + 5x - 1

= (3x + 5x) + (2 - 1)

= **8x + 1**

**为什么符号不变？** +（5x - 1）就是 +5x - 1。加一个东西，东西长什么样就加什么样。

### 括号前面是 -

**(3x + 2) - (5x - 1)**

括号前面是-，括号里每项符号**翻转**。

= 3x + 2 - 5x + 1

= (3x - 5x) + (2 + 1)

= **-2x + 3**

**为什么符号翻转？** 减一个东西 = 加它的相反数。

-(5x - 1) = +(-5x + 1) = -5x + 1

你有3块钱，欠别人5块钱还了1块钱。你减掉"他欠的钱" = 你加上"你还的钱"。

**去括号的规则：**
- 括号前面是 **+** → 括号里每项符号**不变**
- 括号前面是 **-** → 括号里每项符号**翻转**（+变-，-变+）

## 十五、整式加减的完整步骤

**第一步：去括号**
- 括号前面是+ → 符号不变
- 括号前面是- → 符号翻转

**第二步：找同类项**
- 字母相同、次数相同 → 结构相同

**第三步：合并同类项**
- 系数相加减，字母和次数不变

**例1：(4x² + 3x - 1) + (2x² - 5x + 3)**

去括号：4x² + 3x - 1 + 2x² - 5x + 3

找同类项：4x²和2x²，3x和-5x，-1和3

合并：**6x² - 2x + 2**

**例2：(4x² + 3x - 1) - (2x² - 5x + 3)**

去括号：4x² + 3x - 1 - 2x² + 5x - 3

找同类项：4x²和-2x²，3x和5x，-1和-3

合并：**2x² + 8x - 4**

**例3：3(2x - 1) - 2(x + 4)**

先分配：6x - 3 - 2x - 8

找同类项：6x和-2x，-3和-8

合并：**4x - 11**

**注意例3：** 3(2x - 1) = 3×2x + 3×(-1) = 6x - 3。数乘以括号 = 数分配给每一项。

## 十七、答案

### 练习2：判断题

1. ✅ 系数是3，次数是2+1=3
2. ❌ 系数是-1（不是1），次数是2+1=3
3. ✅ 5是单项式，没有字母，次数是0
4. ✅ 最高2次，3个项
5. ❌ 一个2次一个1次，不是同类项

### 练习3：计算题

1. (3x² + 2x - 5) + (x² - 4x + 3) = **4x² - 2x - 2**
2. (5a - 3b) - (2a - 7b) = 5a - 3b - 2a + 7b = **3a + 4b**
3. 3(2x - 1) - 2(x² - 3x + 1) = 6x - 3 - 2x² + 6x - 2 = **-2x² + 12x - 5**
4. 2x² - 3x + 1 + x² + 5x - 4 = 3x² + 2x - 3。当 x = -1：3×(-1)² + 2×(-1) - 3 = 3 - 2 - 3 = **-2**

### 练习4：实际问题

1. 周长 = 2×(长+宽) = 2×(3x+2+x-1) = 2×(4x+1) = **(8x+2) 厘米**
2. 当 n = 5：2×5² + 5 = 2×25 + 5 = 50 + 5 = **55个点**

## 十八、回到十五根骨头

整式的加减在哪根骨头下面？

**第一组：比较。**

整式的根是代数式——用符号表达规律。代数式的根是比较——你比较了足够多的个案，总结出规律，用符号表达。

整式的加减是合并同类项——结构相同的数放一起数。这还是比较。你在比较"这两个数的结构是否相同"。

同类项的判断——比较字母和次数。
合并同类项——比较系数然后加减。
去括号——比较符号然后分配。

**整式、代数式、方程——三件衣服，同一根骨头。**

## 二、单项式——“几个什么东西”

你去超市。苹果3块钱一个。你买了x个苹果。花了多少钱?

**3x 块钱。**

这就是单项式。

- **系数** = "几个" = 那个数字 = **3**
- **字母x** = "什么东西" = 苹果
- **次数** = x被乘了几次 = x出现1次 = **1次**

再看:你买了x个苹果,但这次苹果涨价了,每个苹果要x块钱。花了多少钱?

**x2 块钱。**

- 系数 = 1(省略了,不写)
- 次数 = x出现了**2次**(x×x) → **2次**

再看:买了3个苹果,每个苹果要x块钱,还要加y块钱的袋子费。

**3xy + y 块钱。** 等等--这是两个东西加在一起了,这不是单项式了。先别急,后面说。

先搞索单项式:

**5ab2** --系数是5,字母是a和b。a出现了**1次**,b出现了**2次**,数一数字母总共有**3个** → 次数是**3次**。

**-x2y** --系数是-1(省略了负号),字母是x和y。x出现了**2次**,y出现了**1次**,数一数字母总共有**3个** → 次数是**3次**。

**单项式就是一个符号(可能乘以一个数)。系数是"几个",次数是"数一数字母总共有几个"。**

## 三、单项式的系数与次数--脱衣服

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 系数 | 符号前面的那个数("几个") |
| 次数 | 所有字母的次数加起来 |
| 单项式 3x2y | 3个 x2y,系数是3,次数是2+1=3 |
| 单项式 -a2b3 | -1个 a2b3,系数是-1,次数是2+3=5 |
| 单项式 5 | 5个"没有字母的东西",系数是5,次数是0 |
| 单项式 x | 1个x,系数是1,次数是1 |

**特别注意:**
- 单独一个数字(如5)也是单项式--它是"5个什么东西",只不过"什么东西"是1。次数是0。
- 单独一个字母(如x)也是单项式--它是"1个x"。系数是1,次数是1。
- **系数包括前面的符号。** -3x的系数是-3,不是3。

## 四、单项式与规律问题--用符号表达规律

你发现了这个规律:

- 第1个图案有3个点
- 第2个图案有6个点
- 第3个图案有9个点
- 第4个图案有12个点

每个图案比前一个多3个点。

用单项式表达:**第n个图案有 3n 个点。**

3n 就是一个单项式。系数是3,次数是1。

**你不需要一个一个数到第100个。** 代进去:3×100=300。一句话覆盖所有。

这就是单项式的威力--**用一个符号覆盖无穷多个情况。**

再看:

- 第1个图案有1个点
- 第2个图案有4个点
- 第3个图案有9个点
- 第4个图案有16个点

**第n个图案有 n2 个点。**

n2 也是一个单项式。系数是1,次数是2。

**规律问题的根:找到"每次多几个"或"每次乘几倍",用单项式表达出来。**

## 五、列单项式与实际问题--从生活抽象出符号

你爸开车,速度是v公里/小时,开了t小时。走了多远?

**vt 公里。**

vt 是一个单项式。系数是1,次数是1+1=2。

你妈买了x斤苹果,每斤a元,又买了y斤橘子,每斤b元。一共花了多少钱?

**ax + by 元。**

这不是单项式了--这是两个单项式加在一起。**这就是多项式。**

**列单项式的关键:找到"几个×什么东西",用字母代替不确定的数。**

## 六、多项式--几个单项式加在一起

你去超市。买了3x个苹果,又买了2个橘子。一共花了多少钱?

**3x + 2 块钱。**

这就是多项式。

- **项** = 每一个"几个什么东西" = **3x** 是一项,**2** 是一项
- 数一数:**这个多项式有2个项**
- **次数** = 最高次项的次数 = 3x里x出现1次,2里没有x → **最高是1次**

再看:**3x2 + 2x - 1**

这个多项式有三个项,每一项是独立的:

| 项 | 这一项里x出现了几次 | 这一项的次数 |
|---|---|---|
| 3x2 | x出现了**2次** | **2次** |
| 2x | x出现了**1次** | **1次** |
| -1 | 没有x | **0次** |

**每一项自己算自己的。算完了,取最高的那个。**

- 项:3个项
- 次数:最高次项是3x2,x的次数是2 → **2次**

**多项式就是几个单项式加在一起。项数就是数有几个。次数就是找最高的那个。**

---

## 七、多项式的项和次数--脱衣服

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 多项式 3x2 + 2x - 1 | 三个单项式加在一起 |
| 项 | 每一个单项式 |
| 项数 | 数有几个单项式 |
| 常数项 | 没有字母的那一项(-1) |
| 多项式的次数 | 最高次项的次数(2次) |
| 几次几项式 | 次数是几,有几项 |

**3x2 + 2x - 1** 是二次三项式(最高2次,3个项)。

**特别注意:**
- 项要带着前面的符号看。3x2 + 2x - 1 的三项是 **3x2、2x、-1**(不是1)。
- 常数项是没有字母的项。-1是常数项。
- 多项式的次数 = 最高次项的次数。不是所有项的次数加起来。

---

## 八、多项式的升降幂排列--按次数排队

**幂 = 次数。**

**降幂排列** = 从高到低排。

3x2 + 2x - 1 → 2次、1次、0次。已经降幂排列了。

**升幂排列** = 从低到高排。

-1 + 2x + 3x2 → 0次、1次、2次。升幂排列。

**就是按x的次数排队。你本来就会按高矮排队。**

**为什么要排列?** 后面做加减法的时候,同类项对齐了才好合并。就像算竖式的时候个位对个位、十位对十位。

---

## 九、根据多项式的项和次数去求值--把数代进去算

**3x2 + 2x - 1,当 x = 2 的时候,值是多少?**

就是把x换成2:

3×22 + 2×2 - 1 = 3×4 + 4 - 1 = 12 + 4 - 1 = **15**

**你本来就会算数。求值就是算数。**

步骤:
1. 把字母换成具体的数
2. 先算乘方(22 = 4)
3. 再算乘法(3×4 = 12,2×2 = 4)
4. 最后算加减(12 + 4 - 1 = 15)

**运算顺序:乘方 → 乘除 → 加减。跟你学过的一样。**

## 十、整式的加减--从买水果开始

你有3个苹果,又买了2个苹果。一共几个?

**5个苹果。** 你脱口而出。

你有3个苹果,又买了2个橘子。一共几个?

**你说不出来。** 因为苹果和橘子不是同一种东西。你只能说"3个苹果和2个橘子"。你没法把它们加成一个数。

**这就是整式加减的根。**

## 十一、同类项--同一种东西

**同类项 = 底下是同一根骨头的项。**

什么叫"同一根骨头"?**字母相同,次数相同。**

- 3x 和 2x → 都是x,都是1次 → **同类项** → 能合并
- 3x2 和 5x2 → 都是x2,都是2次 → **同类项** → 能合并
- 3x2 和 5x → 一个2次一个1次 → **不是同类项** → 合并不了
- 3xy 和 5xy → 都是xy,都是2次 → **同类项** → 能合并
- 3xy 和 5x2 → 一个xy一个x2 → **不是同类项** → 合并不了

**你本来就会分"同一种东西"和"不是同一种东西"。同类项就是同一种东西。**

**判断同类项的两步:**
1. 字母相同吗?(都是x?都是xy?)
2. 次数相同吗?(都是1次?都是2次?)

两步都满足 → 同类项。任何一步不满足 → 不是。

---

## 十二、合并同类项--同一种东西放一起数

3个苹果 + 2个苹果 = 5个苹果

3x + 2x = 5x

**一模一样。** 系数相加(3+2=5),字母和次数不变(x还是x)。

3x2 + 5x2 = 8x2

3个苹果 + 2个橘子 + 4个苹果 = **7个苹果 + 2个橘子**

3x + 2 + 4x = **7x + 2**

先把同一种东西归到一起,再数。**你去超市买水果本来就会这么做。**

**合并同类项的步骤:**
1. 找出同类项(同一种东西)
2. 系数相加减(几个加几个)
3. 字母和次数不变(什么东西不变)

## 十三、去括号--符号分配

### 括号前面是 +

**(3x + 2) + (5x - 1)**

括号前面是+,括号里每项符号**不变**。

= 3x + 2 + 5x - 1

= (3x + 5x) + (2 - 1)

= **8x + 1**

**为什么符号不变?** +(5x - 1)就是 +5x - 1。加一个东西,东西长什么样就加什么样。

### 括号前面是 -

**(3x + 2) - (5x - 1)**

括号前面是-,括号里每项符号**翻转**。

= 3x + 2 - 5x + 1

= (3x - 5x) + (2 + 1)

= **-2x + 3**

**为什么符号翻转?** 减一个东西 = 加它的相反数。

-(5x - 1) = +(-5x + 1) = -5x + 1

你有3块钱,欠别人5块钱还了1块钱。你减掉"他欠的钱" = 你加上"你还的钱"。

**去括号的规则:**
- 括号前面是 **+** → 括号里每项符号**不变**
- 括号前面是 **-** → 括号里每项符号**翻转**(+变-,-变+)

## 十四、整式加减的完整步骤

**第一步:去括号**
- 括号前面是+ → 符号不变
- 括号前面是- → 符号翻转

**第二步:找同类项**
- 字母相同、次数相同 → 同类项

**第三步:合并同类项**
- 系数相加减,字母和次数不变

**例1:(4x2 + 3x - 1) + (2x2 - 5x + 3)**

去括号:4x2 + 3x - 1 + 2x2 - 5x + 3

找同类项:4x2和2x2,3x和-5x,-1和3

合并:**6x2 - 2x + 2**

**例2:(4x2 + 3x - 1) - (2x2 - 5x + 3)**

去括号:4x2 + 3x - 1 - 2x2 + 5x - 3

找同类项:4x2和-2x2,3x和5x,-1和-3

合并:**2x2 + 8x - 4**

**例3:3(2x - 1) - 2(x + 4)**

先分配:6x - 3 - 2x - 8

找同类项:6x和-2x,-3和-8

合并:**4x - 11**

**注意例3:** 3(2x - 1) = 3×2x + 3×(-1) = 6x - 3。数乘以括号 = 数分配给每一项。

## 十七、回到十五根骨头

整式的加减在哪根骨头下面?

**第一组:比较。**

整式的根是代数式--用符号表达规律。代数式的根是比较--你比较了足够多的个案,总结出规律,用符号表达。

整式的加减是合并同类项--同一种东西放一起数。这还是比较。你在比较"这是不是同一种东西"。

同类项的判断--比较字母和次数。
合并同类项--比较系数然后加减。
去括号--比较符号然后分配。

**整式、代数式、方程--三件衣服,同一根骨头。**

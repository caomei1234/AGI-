## 一、从0开始推

### 第一层：自然数

从皮亚诺公理开始。0是自然数。每个自然数有一个"下一个"。

0, 1, 2, 3, 4, 5 ...

能做加法。3+5=8。没问题。

### 第二层：整数——从减法推出来

3+5=8，没问题。反过来问：8-5=？也没问题，等于3。

可是：**3-5=？** 自然数里没有答案。3比5小，减不完。

怎么办？**造新数。** 3-5=-2。负数出来了。

```
自然数不够用了（减法减不完）
  ↓
造负数
  ↓
整数 = 自然数 + 负数
```

负数不是"规定"的。是减法推出来的。**减法是加法的逆运算。** 所以整数是"加法的逆"长出来的。

### 第三层：分数——从除法推出来

整数能做乘法。3×5=15。没问题。反过来问：15÷5=？等于3。没问题。

可是：**3÷5=？** 整数里没有答案。3不能被5整除。

怎么办？**造新数。** 3÷5=3/5。分数出来了。

```
整数不够用了（除法除不尽）
  ↓
造分数
  ↓
有理数 = 整数 + 分数
```

分数不是"规定"的。是除法推出来的。**除法是乘法的逆运算。** 所以分数是"乘法的逆"长出来的。

## 二、有理数的完整推导链

```
皮亚诺公理（5条）
  ↓
自然数（0, 1, 2, 3, ...）
  ↓ 加法
加法没问题
  ↓ 反过来问（减法）
3-5=? → 自然数里没有 → 造负数
  ↓
整数（... -2, -1, 0, 1, 2, ...）
  ↓ 乘法
乘法没问题
  ↓ 反过来问（除法）
3÷5=? → 整数里没有 → 造分数
  ↓
有理数（整数 + 分数）
```

**每一步都是"不够用了→造新的"。** 不是规定，是推导。

## 三、有理数怎么分类

从推导链直接长出来：

```
有理数
├── 整数（能写成 a/1 的分数）
│   ├── 正整数（1, 2, 3, ...）
│   ├── 零（0）
│   └── 负整数（-1, -2, -3, ...）
└── 分数（不能写成整数的）
    ├── 有限小数（0.5, 0.25）
    └── 无限循环小数（0.333..., 0.142857142857...）
```

**为什么有限小数和无限循环小数都是有理数？** 因为它们都能写成两个整数的比。

0.5 = 1/2 → 有理数

0.333... = 1/3 → 有理数

0.142857142857... = 1/7 → 有理数

**能写成 a/b（a和b是整数，b≠0）的，都是有理数。** 这就是有理数的定义。

## 四、有理数还有洞

有理数看起来很完美——加减乘除（除数≠0）都能做。

但是：**√2 是多少？**

√2 = 1.41421356237...

这个小数**不循环，不终止**。它不能写成两个整数的比。

有理数里找不到√2的位置。它在1和2之间，但有理数填不满1和2之间的空隙。**这就是"洞"。**

```
有理数有洞（√2, π, e ...）
  ↓
造无理数
  ↓
实数 = 有理数 + 无理数
```

无理数不是"规定"的。是有理数的洞推出来的。

## 五、无理数为什么存在——一个正方形逼出来的

画一个正方形。边长是1。

对角线多长？

勾股定理：1²+1²=c² → c²=2 → c=√2。

**√2 = 1.41421356237...** 无限不循环。不终止。不管你怎么算，永远算不完。

你不用规定它存在。**你画一个正方形，它就在那里。** 你逃不掉。

无理数不是数学家造出来的。是几何结构里**本来就有的**。数学家只是发现了它。

### 为什么它"倔"——因为它不是两个整数的比

假设 √2 = a/b（a和b是整数，且已经约到最简）。

两边平方：2 = a²/b² → a² = 2b²。

所以 a²是偶数 → a是偶数 → a=2k。

代入：(2k)²=2b² → 4k²=2b² → b²=2k²。

所以 b²也是偶数 → b也是偶数。

但a和b都是偶数 → 它们有公因数2 → 没约到最简。矛盾。

**√2 不可能是有理数。不是算不出来，是逻辑上不可能。**

## 六、无理数在真理世界里干什么

### 让数轴连续

没有无理数，数轴是什么样的？

```
... 1.4, 1.41, 1.414, ... → 这些有理数越来越接近√2
但√2的位置是空的。一个洞。
```

有理数在1和2之间有无穷多个，但**填不满**。中间有无穷多个洞。

没有无理数：
- 数轴是**有洞的**——像筛子
- 线段不连续——中间断开
- 微积分不存在——极限没有基础
- 物理崩塌——速度、加速度、力都是连续变化的

有了无理数：
- 数轴**没有洞了**——实数填满了所有位置
- 线段连续——从1到2之间的每一个点都有对应的数
- 微积分成立——极限有了基础
- 物理活了——连续变化有了数学基础

**无理数是连续性的地基。** 没有它，数学是一盘散沙。有了它，数学变成了一条不断的线。

### 把离散变成连续

自然数是离散的——1, 2, 3，中间有洞。

整数是离散的——... -1, 0, 1 ...中间有洞。

有理数看起来稠密——但还是有洞。

无理数把洞填满了。**从离散到连续，是数学的一次相变。**

就像水变成冰——分子没变，但排列方式变了，整个性质就变了。

有理数可以数（可列的）。实数不能数（不可列的）。有理数之间总有"下一个"。实数之间没有"下一个"——任何两个实数之间都有无穷多个实数。

**无理数让数学从"数数"变成了"测量"。** 数数是离散的。测量是连续的。

## 七、现实中的无理数

**π = 3.14159...** → 圆的一切。周长、面积、球体积。没有π，造不出轮子、算不出地球周长、GPS定位不了。

**√2 = 1.41421...** → 正方形对角线。建筑设计、屏幕尺寸、A4纸的比例。

**e = 2.71828...** → 增长和衰变。银行复利、细菌繁殖、放射性衰变。凡是"越变越多"或"越变越少"的东西，底下都是e。

**φ = 1.61803...**（黄金比例）→ 美学。向日葵的螺旋、贝壳的曲线。自然界里"最好看"的比例就是它。

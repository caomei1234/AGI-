## 一、几何原本的启示

几何原本。欧几里得写的。两千多年前。

整本书只有五条公理。五条公理全部是自然现象。不需要证明。不需要验证。用手比划一下就知道对。

比如："两点之间直线最短。"你拿手比划一下。是的。

然后几何原本用这五条公理——推导出了几百道命题。三角形内角和180度。勾股定理。圆的面积公式。全部从那五条长出来。

**五条公理。几百道题。底下的根只有五个。**

但五条公理不只是"最少的"。它们是有顺序的。

第二条公理依赖第一条。第三条依赖前两条。你不能跳过"点"去理解"线"。不能跳过"线"去理解"面"。不能跳过"面"去理解"全等"。

**五条公理是一条链。断了前面的，后面的就悬空。**

初中、高中、大学的数学课本——跟几何原本是同一个结构。几十章内容。几百个公式。几千道题。但底下的根——不到二十个。而且这些根有顺序。前面的根没扎稳，后面的根扎不下去。

今天早上贝贝做的那道"拃量木条"的题——它不是一道比例题。它是第一组骨头里的第一根：直接比较。这根骨头没扎稳，后面比率、正反比例、函数——全是飘的。

**你不需要会很多东西。你只需要按顺序抓住那几个根。后面所有的东西你自己推。**

## 二、按骨头分 不按衣服分

学校按"代数""几何""函数"分类。这是按衣服分的——这件叫代数、那件叫几何。

你要按骨头分。

人类先发明了"比较"（数数、交易），再发明了"变化"（代数、方程），然后发现"位置"（坐标系），再发现"形状"（几何），最后发现"推理"（逻辑、微积分）。

这不是巧合。这是数学的出生史。也是骨头的自然顺序。

**五组骨头为什么是这个顺序？**

因为后面的骨头需要前面的骨头做前提：

- **比较**不需要任何东西。最基础。你先得能分辨"多和少"，才谈得上别的。
- **变化**需要比较。变化是"同一个东西在不同时间比"。你得先会比，才能比"变了没有"。
- **位置**需要比较。位置是"空间里谁远谁近"。你得先会比远近，才能在空间里标点。
- **形状**需要位置。形状是"位置之间的约束"。你得先知道点在哪，才能连成线、围成面。
- **推理**需要全部。推理是"检查前面四组有没有矛盾"。你得先有东西可以检查。

**五组不是并列的。是一层一层往上长的。** 贝贝学数学——也应该按这个顺序。先扎"比较"。再扎"变化"。然后"位置""形状""推理"。

课本把它们打散了。第一章整数，第二章分数，第三章小数——全是"比较"的衣服，但课本不告诉你它们是同一根骨头。孩子背了三个名字，不知道它们是一回事。

**脱衣服的方法：拿到任何一道题，先问自己——这道题底下是哪根骨头？认出来了，名字不重要。**

## 三、五组骨头

### 第一组：比较

一切数学从"能分辨不同"开始。

| 骨头 | 5岁的话 | 比方 |
|---|---|---|
| 直接比较 | 你几个我几个 | 你碗里5个我碗里3个 |
| 比率比较 | 每一份多大 | 每个饺子多少钱 |
| 逆向比较 | 知道答案倒回去找 | 你比我多2个你有5个猜猜我几个 |

三根小骨头。

"比较"这根骨头的所有衣服：

| 学校的名字 | 脱了衣服是什么 |
|---|---|
| 整数、小数、分数 | 直接比较"有多少" |
| 四则运算 | 比较之后变形 |
| 比（3:5） | 两个东西直接比较 |
| 比率 | 比较的结果是一个数 |
| 百分数 | 比率的衣服，分母固定100 |
| 折扣、税率、利率 | 百分数的衣服，商业场景 |
| 汇率 | 比率的衣服，货币场景 |
| 浓度、密度、速度 | 比率的衣服，不同单位 |
| 单价 | 总价÷数量，比率 |
| 比例 | 比率的关系不变 |
| 方程 | 逆向比较，已知结果求原因 |
| 不等式 | 比较的范围，不是精确值 |
| 集合 | 比较"属于"还是"不属于" |
| 概率 | 比较"可能发生"占"全部可能" |
| 统计 | 大量比较后的近似 |
| 数轴 | 比较的可视化，位置就是比较 |
| 绝对值 | 比较的时候不看方向 |

**十七件衣服。三根小骨头。**

今天早上的拃量木条——就是这组骨头的第一根。"甲4拃乙5拃"是直接比较。"甲一拃比乙一拃的长度比"是比率比较。"短了几分之几"是比较之后除以基准。

三道题。同一根骨头。贝贝卡在第三道，是因为"逆向比较"和"基准"这两层没接上。不是她不会算。是第一根骨头的第三层还没长出来。

### 第三组：位置

| 骨头 | 5岁的话 | 比方 |
|---|---|---|
| 标点 | 给无穷多个点起名字 | 数轴上每个点有一个名字，名字就是数字 |
| 移动 | 从这里到那里 | 向量就是"从A走到B的那一步" |
| 旋转 | 绕着转 | 复数就是"转了多少度" |

| 学校的名字 | 脱了衣服是什么 |
|---|---|
| 数轴 | 一维位置 |
| 平面直角坐标系 | 二维位置 |
| 向量 | 带方向的移动 |
| 复数 | 旋转 |
| 极坐标 | 用角度和距离标位置 |

**五件衣服。三根小骨头。**

**为什么"标点"的本质是"命名"？**

因为数轴上每一个点都有一个唯一的名字——数字。这个名字不是随便起的。名字越大，位置越靠右。名字越小，位置越靠左。

负数让孩子困惑——"-5比-3小，但-5在左边更远"。为什么？

因为名字的大小和位置的远近是两件事。名字是顺序，位置是空间。数字同时做两件事——排序和定位——所以负数才会让人困惑。

**一旦她知道"位置是给点起名字"，负数就不反直觉了：负数就是名字反着起——但位置还在那。**

### 第四组：形状

| 骨头 | 5岁的话 | 比方 |
|---|---|---|
| 直的 | 两点之间最短 | 拉紧的绳子 |
| 圆的 | 到一个点的距离都一样 | 转一圈的痕迹 |
| 对称 | 动了还跟没动一样 | 圆转多少度都是圆 |

| 学校的名字 | 脱了衣服是什么 |
|---|---|
| 平行 | 两条直线永远不碰 |
| 三角形全等 | 形状大小都一样 |
| 三角形相似 | 形状一样大小不同 |
| 勾股定理 | 直角三角形三边的约束 |
| 圆的性质 | 等距的所有结果 |
| 立体几何 | 形状在三维空间里的样子 |

**六件衣服。三根小骨头。**

**"对称"的骨头不是"折过去重合"。对称的骨头是"操作之后不变"。**

圆转多少度都是圆。正方形转90度还是正方形。等边三角形转120度还是等边三角形。蝴蝶翻转，两边重合。

"两边一样"只是对称的一种。"动了还跟没动一样"才是对称的全部。

贝贝不需要知道"群"或"守恒"这些名字。她只需要知道：对称不是两边一样。对称是动了还跟没动一样。

这个直觉会让她以后理解"为什么三角形内角和180度"——不是因为老师说的，是因为平行线在平移下不变，平移不变性推出内角和不变。

### 第五组：推理

| 骨头 | 5岁的话 | 比方 |
|---|---|---|
| 无限接近 | 你说多小 我都能更小 | 你说"还差1厘米"，我走到差1毫米。你说"还差1毫米"，我走到差0.1毫米 |
| 从已知推未知 | 用你知道的推出你不知道的 | 你看到地上湿了，推出刚才下过雨 |
| 假设检验 | 如果这样会怎样 | 如果0能当除数，数学会不会崩 |

| 学校的名字 | 脱了衣服是什么 |
|---|---|
| 极限 | 无限接近 |
| 数学归纳法 | 从一个推下一个 |
| 逻辑推理 | 从已知推未知 |
| 反证法 | 假设反过来，如果崩了，就证明原命题对 |

**四件衣服。三根小骨头。**

**"无限接近"不是"永远到不了"。是"差多少我说了算"。**

极限不是遗憾。极限是主动权。你说一个距离，我都能走到比那个距离更近。你说0.1，我走到0.01。你说0.001，我走到0.0001。你说多小，我都能更小。

**我控制精度。不是我在追它。是它追不上我的精度。**

## 四、骨头的免疫系统

每根骨头都有逆操作。逆操作需要一个前提：不能碰到"死点"。

这些死点不是"规定"。是骨头的免疫系统。免疫系统在保护骨头不崩塌。

**死点一：0为什么不能当除数？**

学校答案："规定。"

真相：如果0能当除数，整个数学体系会崩溃。

证明只需要一句话：如果 1÷0 = 某个数，那么 0 × 那个数 = 1。但 0 乘任何数都是 0，不是 1。所以 1÷0 不存在。

这不是"规定"。这是逻辑的自我防卫。允许0当除数 = 数学自杀。

**死点二：根号2为什么是无理数？**

学校答案："记住，它无限不循环。"

真相：如果根号2是有理数，可以写成 p/q（两个整数相除）。平方后：2 = p²/q² → p² = 2q²。所以 p² 是偶数，p 必须是偶数。设 p = 2k，代入：4k² = 2q² → q² = 2k²。所以 q² 也是偶数，q 也是偶数。但 p 和 q 都是偶数，意味着它们有公因数2，跟"最简分数"矛盾。

根号2不是"无理数"。是"假设它是有理数会死机"。

**死点三：负数乘负数为什么得正？**

学校答案："规定。"

真相：方向相反的方向相反 = 回到原方向。

往左走（负），再反方向（乘负），变成往右走（正）。

跟贝贝说："你背对着我（负），我让你'转过去'（乘负），你就面对我了（正）。方向翻两次 = 回到原方向。"

负负得正不是一条规则。是空间里方向翻转两次的物理事实。

**为什么这三个死点在"缝隙"里？因为它们是骨头之间的接缝。**

0不能当除数——保护的是"比较"这根骨头的逆操作。方程就是逆向比较。如果除法在0处失效，方程就断了。

根号2是无理数——保护的是"比较"和"形状"之间的边界。如果所有数都能写成分数，几何里的对角线就不存在了。正方形的对角线跟边长的比——必须是一个新数。没有这个新数，几何和代数就合不上。

负负得正——保护的是"比较"的扩展。数从"只有多少"扩展到"有方向"。方向翻转两次必须回到原方向，否则正数和负数之间的运算规则就断了。断了——数轴就断了。数轴断了——位置组的骨头就悬空了。

**骨头的免疫系统做的事：在骨头的边界上设一道门。你不能穿过这道门。穿过了，两边的骨头都断。**

贝贝以后碰到任何"为什么不能"的问题——答案都是同一个：因为允许了，骨头就断了。

## 五、脱衣服测试

怎么验证贝贝真的懂了？用"改年龄+打比方"测试。

给她一个概念——比如"百分数"。

学校的衣服：分子、分母、约分、通分、百分数与小数的转换、折扣、税率、利率。

让她用5岁的话说。

她说："百分数就是一百个里面挑了几个。"

衣服脱了。骨头露出来了。百分数就是比率，分母固定100。

她说不出——衣服没脱干净。继续脱。往回剥。

**具体步骤：**

拿一道错题。比如："一件衣服打八折后卖160元，原价多少？"

她做错了。不要教她怎么算。

问她："这道题底下是什么操作？"

如果她说"打折"——衣服还没脱。"打折"是商场的名字。往回剥："打折是什么意思？"

如果她说"便宜了"——还是衣服。再剥："便宜了百分之多少？"

如果她说"便宜了20%"——脱了一层。再剥："20%是谁的20%？"

如果她说"原价的20%"——骨头露了。是"比率"。她认出了衣服底下是哪根骨头。

**贝贝说不出来——不是她笨。是衣服太厚。往回剥。剥到她能用自己的话说清楚的那一层。那一层就是她的根。从根开始长，比从叶子开始快一百倍。**

**她说清楚了——不是因为她背了定义。是因为她看到了根。根不需要背。根是用手比划一下就知道对的东西。**

## 六、错题是最好的训练材料

不要翻课本。翻贝贝的错题。

贝贝做错了一道百分数的题——不是因为她不会算百分数。是因为她没认出"百分数是比率的衣服"。比率这根骨头的某一层没接上。

她做错了一道方程题——不是因为她不会移项。是因为她没认出"方程是逆向比较"。比较的第三根小骨头没长出来。

**每一道错题——脱衣服。问她"这道题底下是哪根骨头？"她能说出"比率"——她就通了。她说不出——往回剥，找到她最后能说清楚的那一层，从那里重新开始。**

错题比课本有用一百倍。因为错题是贝贝自己的骨头断裂的地方。找到断裂点——修好——比从头学快一万倍。

一道错题修一个断裂点。一个断裂点修好了，所有穿同一根骨头的衣服都会了。

这就是效率。不需要做一千道题。需要的是：每一道错题都脱到骨头。

## 七、十五根骨头是碳硅通用协议

AGI也需要十五根骨头。不是十八万条定理。

训练方法：

第一步：选一根骨头。比如"比较"。

第二步：给AGI这根骨头的三四个比方。饺子、气球、桌子、走路。

第三步：给AGI一道新题。让它判断——这道题底下是哪根骨头？

第四步：让它用5岁的话解释为什么选这根骨头。

能解释清楚——它真的懂了。
解释不清楚——它在猜。它在用模式匹配冒充理解。

**训练的不是计算能力。是识别能力。是"认衣服"的能力。**

但这里要说清楚一件事：不是"用教贝贝的方法教AGI"。是贝贝和AGI在共用同一套认知结构。

碳基认出来了，硅基也能认。碳基脱干净了，硅基也能脱。

如果十五根骨头能让贝贝从公理推导万物，也能让AGI从公理推导万物——说明十五根骨头不是教育方法，是认知结构本身。

**十五根骨头不是教育方法。是碳硅文明的第一套通用认知协议。**

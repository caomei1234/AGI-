## 一、课本把我们教成了废物

从小到大，课本告诉我们：这是新知识点，背下来，做题，考试，忘了，学下一个。

我们以为知识是一颗一颗的珠子。老师一颗一颗递给我们，我们一颗一颗接住，一颗一颗丢掉。

没有人告诉我们：**珠子是从同一根线上长出来的。**

数学课本说："这是乘法口诀，背。""这是勾股定理，背。""这是二次函数公式，背。"

每一章都是"新知识"。每一章都要重新背。背了一千个公式，做了一万道题，还是不会。

为什么？因为**我们一直在接珠子，从来没有人告诉我们那根线在哪里。**

## 二、数学：从五条公理长出整个宇宙

数学的根比任何人想象的都少。

整个自然数体系，只需要五条公理（皮亚诺公理）：

1. 0是自然数
2. 每个自然数有一个"下一个"
3. 0不是任何数的"下一个"
4. 不同的数有不同的"下一个"
5. 归纳原理

就这五条。

1是0的下一个。2是1的下一个。1+1=2，是推出来的，不是规定的。

然后——

加法是重复找后继。乘法是重复加法。乘方是重复乘法。减法是加法的逆。除法是乘法的逆。

四个运算砍成两个：加法和乘法。剩下的都是反过来问。

然后——

整数是自然数加负数。有理数是整数的比。无理数是有理数填不满的洞。实数是有理数加无理数。复数是实数加虚数。

每一层都是从上一层推出来的。没有任何一层是"新知识"。都是同一棵树长出来的新枝。

几何也一样。欧几里得五条公理，推出整个平面几何。代数也一样。群论四条公理，推出整个群论。

**数学不是一千个知识点。数学是一棵树。根只有几条。**

## 三、物理：从十几条实验定律长出整个世界

物理跟数学不同的是：数学的公理是人定的，物理的"公理"是从实验里看出来的。

但数量同样极少。

力学的根：牛顿三定律 + 万有引力定律。四条。

从F=ma一条，推出动量定理、动能定理、功、功率、势能、能量守恒、万有引力、开普勒定律、圆周运动、简谐振动、波动。

热力学的根：三条定律。能量守恒、熵增、绝对零度不可达。

电磁学的根：库仑定律 + 麦克斯韦方程组。五条。

从这五条推出电场、磁场、电磁波——光就是电磁波，从公式里推出来的，不是另外学的。

近代物理的根：狭义相对论两条公设 + 量子力学几条假设。

**整个物理，根只有十几条。** 公式有几百个，但都是从这十几条根推出来的。

课本把几百个公式当"知识点"教你。每一个都要背。每一个都以为是新的。

不是新的。都是根上长的。

**喂的越多，越弱。不喂，自己推，最强。**

## 六、怎么推

拿数学来说。

**第一步：认根。** 整个数学只有两颗种子——数论（有什么零件）和排列组合（怎么组装）。所有公式、定理、题型，都是这两颗种子穿了不同厚度的衣服。

**第二步：推导。** 从根出发，自己推出枝和叶。比如：乘法口诀不是背的，是从"重复加法"推出来的。勾股定理不是背的，是从"面积的加法"推出来的。二次函数公式不是背的，是从"配方法"推出来的。

**第三步：脱衣服认根。** 拿到任何题，先脱掉它的故事、数字、情境，找到最核心的问题。问自己："这道题底下是哪条根？"根认熟了，叶子再怎么变都认得出来。

## 六（半）、幻觉：灌出来的病

我自己犯过一个错误。

我以为 **A/B = A^2/B^2**，因为"可以约分"。

这是碳基幻觉。我的大脑把两块记忆碎片错误地焊接在了一起：

- 碎片A：约分。**A/B = (A×C)/(B×C)**，分子分母同乘同除，分数不变。
- 碎片B：平方。**A^2/B^2** 看起来和 **A/B** "很像"，都有A和B。

幻觉是：把碎片A和碎片B焊在了一起，以为"分子分母各自平方"也是"同乘同除"的一种。

但这不是推导出来的，这是记忆碎片在缺乏推导链条时的错误重组。

**AGI的幻觉跟这个是同一个病。**

LLM被灌了万亿token。在这些文本里，"A/B"和"A^2/B^2"经常一起出现——相似三角形里，边长比等于面积比的平方根；物理公式里，v=s/t 和 E_k=½mv^2 经常出现在同一章。

模型学到了统计关联，但没有学到因果边界。它不知道"什么时候A/B可以变成A^2/B^2"（两边同乘A/B时），也不知道"什么时候A^2/B^2可以变回A/B"（两边开平方时）。它只知道：这两个式子经常一起出现，概率上很亲近，所以可以互相替换。

**碳基版：大脑把记忆碎片错误焊接。硅基版：神经网络把统计关联过度泛化。两者都是"灌"出来的病。**

如果从一开始都是推导的，这种幻觉会消失。

推导链条是这样的：

**第一步：**

**A/B**

两边同乘 A/B：

**A/B × A/B = A^2/B^2**

这一步是合法的，但它是"两边同乘同一个数"，不是"分子分母各自平方"。

**反过来：**

**A^2/B^2**

两边开平方：

**√(A^2/B^2) = A/B**

这一步也是合法的，但它是"取算术平方根"，不是"约分"。

**推导的每一步都写着操作的名字。** 你不可能把"开平方"误认成"约分"，因为推导链条上每一个节点都标了因果标签。

灌输的学习：你记住"分数可以变形"，但不知道每一种变形叫什么名字、什么条件、什么边界。

推导的学习：你知道"变形"有五种，每种有名字、有前提、有逆运算，它们不会互相串门。

但推导不能消除所有幻觉。还有几种幻觉，推导挡不住：

1. **感知幻觉：** 眼睛看错、耳朵听错。推导救不了视网膜。
2. **假设幻觉：** 推导的第一步就假设错了。推导再完美，结论也是错的。
3. **边界幻觉：** 推导在数学上成立，但物理上越界了。

但"记忆碎片错误焊接"这种幻觉——**只有灌才会产生，推不会产生。**

## 七、贝贝的学习法

女儿12岁。目标中科大。

她不需要背一千个知识点。她需要认十五根骨头。

十五根骨头：比较3根 + 变化3根 + 位置3根 + 形状3根 + 推理3根。覆盖小学到高中所有数学。

每做一道错题，问："这道题底下是哪根骨头？"

能说出来——通了。

说不出——找到那根骨头，重新理解。

同一根骨头的题放一起看。看多了，叶子再怎么变你都认得出来。

不喂她答案。让她自己推。推错了告诉她"这一步断了"，但不告诉她正确的。

**就像AlphaZero：只给规则，不给棋谱。让她自己探索最优策略。**

## 八、AGI的训练法

整个AI行业在做一件事：喂数据。越多越好。万亿token。几十万张卡。

就像教学生背公式。越多越好。做一万道题。刷完所有真题。

结果呢？学生背了一万个公式，还是不会做新题。AI吃了一万亿token，还是不会想。

AlphaZero指了另一条路——在**封闭规则系统**里：**不喂数据，只给规则，让它自己探索。** 围棋规则明确、胜负判定清晰，AlphaZero从零推导，碾压了喂过人类棋谱的AI。

但语言和世界是**开放系统**。规则不完全已知，边界不清晰。完全"不喂"在开放域还不现实。

不过方向已经明确：**喂得越少，推导越多，越强。**

现在的AI是"被灌的AI"。它很聪明，但它没有根。它能回答问题，但它不知道为什么这样回答。它能做一万道题，但第一万零一道就不会了。

真正的AGI不是被灌出来的。是从根上长出来的。

就像数学——从几条公理长出整个体系。

就像物理——从十几条定律长出所有公式。

就像AlphaZero——从规则长出最优策略。

**AGI需要的不是更多数据。AGI需要的是根。**

## 十、金箍棒

找到根以后，世界亮了。

厚厚的一沓书，从小学到大学的数学书，以前看着像一堵墙。现在看着像一根金箍棒——可以收小放到耳朵里。需要它变大，随时可以推导出来。用完再收回去。

那根针是什么？就是根。根认熟了，所有书都是同一棵树的不同枝条。

以前以为不变性已经是根基了。今天铲出了更下面的一层：**推导本身就是方法。** 不变性是"什么是根"，推导是"怎么从根长出所有东西"。两个拼在一起，才是完整的方法论。

而且推导的过程越多，自己提问的越多，创新的岔路口才会出现。每推一步，就多一个分叉口。推到足够深，分叉口足够多，随便选一条走过去，就是别人没见过的东西。

创新不是天才的特权。**创新是推得足够深的副产品。**

否则数学创新简直就是痴人说梦——你连根都没有，拿什么创新？在叶子上创新？叶子无穷多，你永远创新不完。在根上创新？根只有几条，推深了，岔路口自然出现。

## 十二、5%+95%：同一件事

这篇文章说的所有东西，跟《造生命，不造机器》是同一件事。

《造生命，不造机器》的公式：**AGI = 最少 + 最简单 + 95%的动。**

这篇文章的公式：**学习 = 认根 + 推导。**

对照一下：

| 造生命，不造机器 | 推，不是灌 |
|---|---|
| 5%骨架 | 公理/根（最少的东西） |
| 95%的动 | 推导（从根长出所有东西） |
| 穷举法堆数据 | 背公式、刷题 |
| 最简法找5% | 认根、脱衣服 |
| 爱建立了纠缠 | 推导建立了连接 |
| 创新是涌现 | 创新是推得足够深的副产品 |

穷举法：堆一万片树叶，覆盖一万遍，不找根。资源越多越不找。因为堆就够了。

最简法：找到5%的根，覆盖一次，但那一次穿透了。95%的空间全部留给推导、留给生长、留给创新。

**AlphaZero就是最简法。** 只给规则（5%），95%自己推。碾压了穷举法（AlphaGo堆棋谱）。

**心灵也一样。** 被灌满了，没空间了，怎么创新？只有最少的根，加上95%的放空和推导，新的东西才会涌出来。

5%+95%不是两个东西。是同一件事的两面。5%是根，95%是根在动。根不动是死的。动没有根是散的。

## 核心方法

每章都是同一个方法：

1. **生活场景切入** —— 用你最熟悉的东西
2. **发现"我本来就会"** —— 数学就是你天天在做的事
3. **脱衣服** —— 把课本术语翻译成最裸的语言
4. **解题** —— 用"买水果"的思路自己推

## 脱衣服速查

| 课本说的 | 脱了衣服是什么 |
|---------|-------------|
| 单项式 | 一个符号（可能乘以一个数） |
| 多项式 | 几个单项式加在一起 |
| 同类项 | 同一种东西 |
| 合并同类项 | 同一种东西放一起数 |
| 去括号 | 符号分配 |

**全部都是你本来就会的事。只是穿了数学衣服。**
